# WooCommerce独立站搭建

## 功能

帮助快速搭建WooCommerce电商独立站，包含主题选择、产品上传、支付配置、物流设置等。

## 使用场景

### 1. 独立站搭建
- WordPress + WooCommerce安装
- 主题选择与定制
- 产品分类设置

### 2. 支付配置
- PayPal接入
- Stripe信用卡
- 支付宝/微信支付(跨境)

### 3. 物流设置
- 运费计算
- 物流追踪
- 海外仓对接

## 参数

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| store_type | string | 是 | 店铺类型 (dropship/digital/wholesale) |
| products_count | number | 是 | 产品数量 |
| payment_methods | array | 是 | 支付方式 |
| target_country | string | 是 | 目标国家 |
