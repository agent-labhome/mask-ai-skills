# CherryStudio AI生产力工作室

## 描述

CherryStudio - 2026年热门AI生产力工作室，支持300+ AI Assistants。适用于：(1) 多模型切换 (2) 助手管理 (3) 工作流自动化 (4) 企业协作 (5) 本地部署

## 指令

### 1. 核心功能
- **多模型支持**: GPT-4, Claude, Gemini, Llama, Mistral等
- **助手市场**: 300+预建助手模板
- **对话管理**: 会话组织和历史记录
- **文件处理**: PDF, Word, Excel, 图片分析
- **插件扩展**: 第三方集成

### 2. 安装配置
```bash
# 桌面应用
下载安装包: https://cherrystudio.ai

# 或使用Docker
docker pull cherrystudio/cherrystudio
docker run -d -p 8080:8080 cherrystudio/cherrystudio
```

### 3. 助手使用
从助手市场选择：
- **写作助手**: 文章、博客、营销文案
- **编程助手**: 代码审查、调试、优化
- **数据分析**: Excel分析、图表生成
- **翻译助手**: 多语言互译
- **设计助手**: UI/UX建议

### 4. 企业部署
- **私有云**: Docker/Kubernetes部署
- **SSO**: 企业身份集成
- **API**: 开放API供内部系统调用
- **审计**: 完整使用日志

### 5. 工作流自动化
创建自动化工作流：
- 定时任务
- 条件触发
- 多助手协作
- webhook集成

## 最佳实践

1. **助手选择**: 根据任务选择最佳助手
2. **提示优化**: 提供清晰上下文
3. **结果验证**: 重要内容人工复核
4. **团队共享**: 共享成功的工作流
