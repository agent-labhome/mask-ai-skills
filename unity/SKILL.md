# SKILL.md: Unity

## Metadata
- **Category**: Game Development, Software Engineering
- **Type**: Game Engine, Development Platform
- **Experience Level**: Junior to Senior
- **Related Skills**: C#, 3D Math, Game Design, Shader Programming, Physics, Animation, UI/UX
- **Common Roles**: Unity Developer, Gameplay Programmer, Technical Artist, Tools Developer

## Overview
Unity is a cross-platform game engine and development environment used for creating 2D, 3D, VR, and AR experiences. It features a component-based architecture, extensive asset pipeline, and supports C# scripting.

## Core Competencies

### 1. **Engine Fundamentals**
- **Component System**: MonoBehaviour, GameObjects, Prefabs
- **Scene Management**: Loading, additive scenes, scene transitions
- **Asset Pipeline**: Import settings, asset bundles, addressables
- **Editor Extensions**: Custom inspectors, editor windows, property drawers

### 2. **Programming & Scripting**
- **C# Proficiency**: Async/await, LINQ, events/delegates
- **Unity APIs**: Time, Input, Physics, Coroutines
- **Architecture Patterns**: ECS (Entities, Components, Systems), MVC, Observer
- **Performance**: Object pooling, garbage collection management, profiling

### 3. **Graphics & Rendering**
- **Materials & Shaders**: ShaderGraph, HLSL/ShaderLab
- **Lighting**: Global illumination, light probes, reflection probes
- **Post-processing**: Volume system, custom effects
- **Optimization**: Draw calls, batching, LODs, occlusion culling

### 4. **Physics & Animation**
- **Physics Systems**: Rigidbody, colliders, joints, raycasting
- **Animation**: Animator Controller, blend trees, state machines
- **Timeline**: Cinematic sequences, animation tracks
- **DOTS Physics**: For high-performance simulations
