# 投资顾问

## 功能

专业的投资建议工具，配置、风险、组合全覆盖。

## 使用场景

### 1. 资产配置

**输入:** 资金 + 风险偏好

**输出:**
- 配置建议
- 各资产比例
- 入场时机
- 再平衡策略

### 2. 风险评估

**输入:** 投资组合

**输出:**
- 风险等级
- 最大回撤
- 波动率
- 建议

### 3. 投资组合优化

**输入:** 现有持仓

**输出:**
- 优化建议
- 分散度分析
- 相关性分析
- 调整方案

### 4. 市场分析

**输入:** 市场环境

**输出:**
- 趋势判断
- 机会/风险
- 策略建议

## 参数

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| task | string | 是 | 任务类型 |
| capital | number | 是 | 资金量 |
| risk | string | 是 | 风险偏好 |
