## Purpose
Quickly generate visual concepts from a brief description. Turns text into simple, clear sketches to communicate ideas visually.

## Usage
```
/sketch [description]
```
Example: `/sketch a friendly robot holding a coffee cup`

## Key Features
- **Simple line drawings** - Clean, minimal sketches
- **Concept visualization** - Great for brainstorming and ideation
- **Rapid iteration** - Get visual ideas in seconds
- **Universal communication** - Visuals that transcend language barriers

## Best For
- Brainstorming sessions
- Explaining complex ideas simply
- UI/UX wireframing concepts
- Product design ideation
- Storyboarding initial frames
- Visual note-taking
