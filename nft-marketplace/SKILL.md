# Nft Marketplace

## 功能
- 自动执行任务
- 提供专业建议
- 生成优化方案

## Core Competencies
- **Smart Contract Development**: Solidity, ERC-721/1155 standards, OpenZeppelin libraries
- **Blockchain Integration**: Ethereum/Polygon/Solana wallet connections (MetaMask, Phantom)
- **Frontend Development**: React/Next.js with Web3.js or Ethers.js integration
- **IPFS/Decentralized Storage**: Filecoin, Arweave, or Pinata for NFT metadata
- **Marketplace Features**: Listings, auctions, bids, collections, royalties

## Key Skills
- Wallet authentication and transaction handling
- Gas optimization and contract security
- Responsive NFT gallery and detail views
- Real-time price/status updates
- Creator royalty implementation (EIP-2981)

## Technical Stack
**Common**: React, Node.js, Hardhat/Truffle, IPFS, GraphQL/The Graph (for indexing)
