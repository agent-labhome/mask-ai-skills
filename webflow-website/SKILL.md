## Purpose
Build and manage responsive websites using Webflow's visual development platform.

## Capabilities
- Create responsive website layouts and components
- Implement Webflow CMS structures and dynamic content
- Configure interactions, animations, and custom code
- Set up hosting, domains, and SEO settings
- Export clean, production-ready code when needed

## Best Practices
- Use semantic HTML structure and proper naming conventions
- Implement mobile-first responsive design
- Leverage Webflow's CMS for scalable content management
- Maintain clean class naming and reusable components
- Optimize images and assets for performance

## Limitations
- Requires Webflow account for full functionality
- Complex backend logic may need custom code
- Some advanced features require higher-tier plans
