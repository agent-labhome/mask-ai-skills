# Micro Saas

## 功能
- 自动执行任务
- 提供专业建议
- 生成优化方案

## Core Competencies

### Technical Stack
- **Backend**: Node.js/Python (FastAPI/Flask), Go (for performance-critical services)
- **Frontend**: React/Vue.js with TypeScript, Tailwind CSS
- **Database**: PostgreSQL, SQLite (for embedded), Redis (caching)
- **Infrastructure**: Docker, AWS/Azure/GCP basics, serverless (AWS Lambda, Vercel)
- **APIs**: REST, GraphQL, webhooks integration

### Product Development
- MVP development (3-6 month cycles)
- User onboarding flows
- Subscription management (Stripe/Paddle)
- Analytics implementation (Mixpanel, Amplitude basics)
- Feature flagging (LaunchDarkly alternatives)

### Business Fundamentals
- Pricing strategy (freemium, tiered, usage-based)
- Customer acquisition cost (CAC) calculation
- Lifetime value (LTV) estimation
