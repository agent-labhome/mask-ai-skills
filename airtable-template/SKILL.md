# SKILL.md: Airtable Template Brief

## Purpose
This skill enables the creation of structured, professional briefs using Airtable as a template system. It transforms project requirements into organized, actionable briefs with clear sections, deliverables, and stakeholder information.

## Core Capabilities
- **Template-Based Brief Creation**: Uses Airtable's structured format to generate consistent briefs
- **Project Requirement Analysis**: Extracts key information from conversations to populate brief sections
- **Stakeholder Management**: Identifies and documents all involved parties and their roles
- **Deliverable Specification**: Clearly defines what needs to be created or achieved
- **Timeline Structuring**: Organizes project phases and deadlines

## Input Processing
The skill analyzes conversations to identify:
- Project objectives and goals
- Target audience or users
- Key stakeholders and decision-makers
- Required deliverables and formats
- Success criteria and metrics
- Budget and resource constraints
- Timeline expectations
