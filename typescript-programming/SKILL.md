## Best Practices
- Use strict mode (`strict: true`)
- Prefer `interface` for object shapes, `type` for unions/aliases
- Avoid `any` - use `unknown` or proper types
- Leverage utility types (`Partial`, `Pick`, `Omit`)
- Write declaration files for libraries
