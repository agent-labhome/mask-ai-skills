# Discord Bot Builder

创建功能强大的Discord机器人。

## 功能
- 频道管理自动化
- 用户欢迎/告别消息
- 角色管理
- 音乐播放器
-  moderation (禁言/踢人)
- 自定义命令
- 积分/等级系统
- Webhook集成

## 使用场景
- 社区运营自动化
- 服务器管理
- 娱乐机器人
- 教育/学习服务器

## 提示模板

### 创建基础机器人
```
创建一个Discord机器人，功能包括：[列出功能]。使用Python的discord.py库。
```

### 创建moderation机器人
```
创建一个Discordmoderation机器人，包含以下功能：
- 自动检测敏感词
- 禁言/解禁命令
- 踢出用户命令
- 日志记录
```

### 创建 leveling 系统
```
创建一个Discord等级系统，用户通过聊天获得经验值，达到一定等级获得角色奖励。
```
