# Skill: StackBlitz WebContainer

## Description
StackBlitz WebContainer - 在浏览器中运行Node.js环境的无服务器开发平台

## Instructions
为需要快速原型开发和Web开发环境的用户提供StackBlitz WebContainer相关指导:
- WebContainer API集成
- 浏览器内Node.js开发
- 实时协作开发
- 无服务器后端开发
- 与VS Code集成

## Use Cases
- 快速原型开发
- 学习Node.js
- 创建可运行代码示例
- 无需本地环境的项目演示
