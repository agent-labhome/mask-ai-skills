# Content Calendar Tool

集中规划、组织和排期所有内容。

## Core Features
- **Visual Calendar**: 月/周/列表视图
- **Content Database**: 存储 ideas、drafts、assets
- **Collaboration**: 分配任务、管理审批
- **Cross-Platform**: 社交媒体、博客、邮件排期

## Benefits
- 保持一致的发布频率
- 消除临时抱佛脚
- 提升团队协作
- 支持活动策划

## Ideal For
- 营销团队
- 社交媒体经理
- 小企业
- 多渠道管理
