## Core Purpose
Create a cohesive, scalable visual language that ensures consistency across all products while enabling efficient development.

## Key Responsibilities

### 1. Foundation Design
- Establish color palettes with semantic meaning
- Define typography hierarchy and scale
- Create spacing system (grid, margins, padding)
- Design iconography and illustration style

### 2. Component Library
- Build reusable UI components (buttons, forms, cards, etc.)
- Document component states and behaviors
- Ensure accessibility compliance (WCAG 2.1+)
- Create responsive design patterns

### 3. Documentation & Governance
- Maintain design tokens and variables
- Create usage guidelines and best practices
- Establish contribution and versioning processes
- Document design principles and decision rationale
