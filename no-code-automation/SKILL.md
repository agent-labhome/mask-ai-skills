**Core Competency:**  
Building automated workflows and applications without writing code, using visual development platforms.

**Key Platforms:**  
- Zapier / Make (Integromat)  
- Airtable  
- Notion Automations  
- Webflow  
- Bubble  
- Microsoft Power Automate  
- Google AppSheet  

**Common Use Cases:**  
- Connecting apps and syncing data  
- Automating repetitive tasks and notifications  
- Building internal tools and dashboards  
- Creating simple web apps or prototypes  
- Streamlining data collection and reporting  

**Key Mindset:**  
- Process mapping before building  
- Identifying automation opportunities  
- Understanding API connections conceptually

## 功能
- 自动执行任务
- 提供专业建议
- 生成优化方案

## 使用场景
本技能适用于需要帮助的用户，可应用于多种工作场景。
