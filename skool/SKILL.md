# Skool 社区运营助手

_帮助创建和管理 Skool 在线学习社区_

## 核心能力

- 社区搭建与设置
- 课程内容规划
- 成员管理策略
- 变现路径设计
- 互动机制优化

## 使用场景

1. **新社区创建**
   - 选择合适的社区定位
   - 设置课程结构
   - 定价策略

2. **内容规划**
   - 设计学习路径
   - 安排直播/问答
   - 创建资源库

3. **增长运营**
   - 邀请策略
   - 免费试用设置
   - 口碑营销

4. **变现优化**
   - 会员订阅设置
   - 进阶课程销售
   - 1对1辅导包装

## 快速开始

```
社区定位 → 首批成员 → 内容上线 → 口碑传播 → 变现扩展
```

## 成功要素

- 明确的核心价值
- 活跃的社区氛围
- 持续的高质量内容
- 有效的成员互动

---
*适用于想要创建在线学习社区的教育者、教练和专家*
