## Metadata
- **Category**: Mobile Development
- **Experience Level**: Junior to Senior
- **Type**: Framework/Library
- **Prerequisites**: JavaScript/TypeScript, React fundamentals, Node.js/npm
- **Platform**: iOS, Android, Web (with additional libraries)
- **Release**: 2015 (Facebook/Meta)

## Core Competencies

### 1. Fundamentals
- **React Concepts**: Components, JSX, props, state, hooks (useState, useEffect, useContext, etc.)
- **RN Components**: View, Text, Image, ScrollView, TextInput, TouchableOpacity
- **Styling**: StyleSheet API, Flexbox layout, platform-specific styles
- **Navigation**: React Navigation (Stack, Tab, Drawer), deep linking
- **Platform APIs**: Accessing native device features (camera, location, notifications)

### 2. Development Setup
```bash
# Quick start
npx react-native init MyApp
cd MyApp
npx react-native run-ios
npx react-native run-android

# Expo alternative
npx create-expo-app MyApp
cd MyApp
npx expo start
```
