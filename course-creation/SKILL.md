## Purpose
Create structured, effective online courses from any topic or content.

## Core Process
1. **Analyze** - Break down source material into core concepts
2. **Structure** - Organize into logical learning progression
3. **Develop** - Create engaging content with clear learning objectives
4. **Validate** - Ensure completeness and pedagogical soundness

## Key Principles
- **Chunking**: Break content into digestible modules (15-30 min each)
- **Scaffolding**: Build from simple to complex concepts
- **Engagement**: Mix theory with practical application
- **Assessment**: Include knowledge checks and practical exercises
