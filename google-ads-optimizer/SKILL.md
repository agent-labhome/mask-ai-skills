# Google Ads Optimizer

## Quick Start

### 分析现有广告系列

提供广告系列ID或账户数据，分析以下内容：
- 关键词表现（CTR、CPC、转化率）
- 广告组健康度
- 预算分配建议
- 竞争环境分析

### 创建新广告系列

1. 确定营销目标（品牌认知/流量/转化/再营销）
2. 研究目标关键词（搜索量、竞争度、相关性）
3. 构建关键词列表（广泛/精确/短语匹配）
4. 撰写广告文案（标题1-15字符、描述1-90字符）
5. 设置出价策略（手动CPC/智能出价/目标ROAS）
6. 配置转化跟踪

## Keyword Strategy

### 匹配类型选择

| 类型 | 适用场景 | 风险 |
|------|----------|------|
| 广泛匹配 | 扩大覆盖面 | 可能带来不相关流量 |
| 短语匹配 | 平衡覆盖和精准 | 需要持续优化 |
| 精确匹配 | 高转化意图 | 搜索量有限 |

### 关键词研究要点

- 优先长尾关键词（3-5词）
- 排除不相关关键词（否定关键词）
- 关注搜索量>1000的词
- 分析竞争对手关键词

## Ad Copy Optimization

### 标题最佳实践

```
标题1: 品牌名 + 核心产品/服务
标题2: 核心价值主张
标题3: 行动号召 + 紧迫感
标题4-15: 补充信息（可选）
```

### 描述要点

- 第一行：价值主张
- 第二行：社会证明/评分
- 第三行：行动号召
- 第四行：促销/紧迫感

### 高点击率技巧

1. 包含数字（"节省50%"）
2. 使用行动词（"获取"、"免费试用"）
3. 突出差异化（"唯一"、"顶级"）
4. 添加促销（"限时"、"优惠"）

## Bidding Strategies

### 出价策略对比

| 策略 | 目标 | 适用阶段 |
|------|------|----------|
| 手动CPC | 控制成本 | 测试期 |
| 智能出价 | 转化最大化 | 稳定期 |
| 目标ROAS | 投资回报率 | 成熟期 |
| 目标CPA | 单次转化成本 | 转化优化 |

### 预算分配

- 测试期：每个广告组日预算$10-20
- 稳定期：基于CPA×转化数×1.5
- 旺季：提前2周调整预算+20%

## Conversion Tracking

### 必需跟踪的转化

1. 购买/下单
2. 表单提交
3. 电话拨打
4. 注册/订阅
5. 视频观看

### 设置流程

1. 创建转化操作
2. 安装跟踪代码（全局站点代码）
3. 验证跟踪正常
4. 导入转化数据到广告系列

## Performance Analysis

### 关键指标

| 指标 | 优秀 | 良好 | 需优化 |
|------|------|------|--------|
| CTR | >5% | 2-5% | <2% |
| 质量得分 | 8-10 | 5-7 | 1-4 |
| 转化率 | >5% | 1-5% | <1% |
| ROAS | >4 | 2-4 | <2 |

### 优化频率

- 日常：检查预算和出价
- 每周：分析关键词表现，调整出价
- 每月：全面审查广告系列结构
- 季度：评估整体策略

## Common Issues

### 点击率高但无转化

- 检查落地页相关性
- 审查转化跟踪设置
- 分析搜索词报告，添加否定关键词
- 评估广告承诺与实际一致性

### 成本过高

- 降低CPC出价
- 收缩关键词匹配类型
- 添加否定关键词
- 优化广告质量得分

### 展示量不足

- 放宽关键词匹配
- 提高出价
- 检查账户级别预算
- 审查受众定位是否过窄
