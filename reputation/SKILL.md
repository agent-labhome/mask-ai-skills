# Reputation Management SKILL

## Purpose
Track and manage reputation scores for users based on their actions and contributions.

## Core Functions
- **Increment/Decrement**: Adjust reputation with optional reasons
- **Get/Set**: Retrieve or manually set reputation values
- **Leaderboard**: View top users by reputation
- **History**: Track reputation changes over time

## Key Features
- Persistent storage across sessions
- Configurable cooldowns to prevent spam
- Action-based reputation (helpful answers, quality contributions, etc.)
- Optional moderation tools for manual adjustments

## Typical Use Cases
- Reward helpful community members
- Discourage negative behavior
- Gamify participation and quality contributions
- Identify trusted users and subject matter experts
