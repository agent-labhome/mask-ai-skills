# NotebookLM

NotebookLM是Google的AI笔记和研究助手，可以将文档转换为问答、摘要和播客。

## 核心功能

### 1. 文档处理
- 导入PDF、Google Docs、网页、音频
- 自动提取关键信息
- 跨文档关联分析

### 2. 问答系统
- 基于_sources回答问题
- 引用原文溯源
- 多文档联合查询

### 3. 音频概览
- 将文档转为AI播客
- 双AI主持人对话
- 学习/通勤场景

## 与OpenClaw协作

### OpenClaw → NotebookLM
```
1. 抓取目标网站内容
2. 自动导入NotebookLM
3. 清洗去重
4. 创建sources
5. 生成问答对
```

### 自动化场景
- 研究竞品：自动抓取并导入
- 学习文档：批量导入+摘要
- 内容创作：素材收集+整理

## 使用技巧

1. **_sources原则**：只添加相关源，AI只基于这些回答
2. **逐步导入**：一次不要超过20个sources
3. **定期清理**：移除过时sources保持准确性
4. **利用音频**：通勤时听AI播客

## 最佳实践

- 研究场景：先批量导入，再深度问答
- 学习场景：生成音频摘要随身听
- 创作场景：作为灵感来源和事实核查工具
