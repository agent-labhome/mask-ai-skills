## 使用示例

```
用户: 我想优化LinkedIn profile

助手:
## 💼 LinkedIn优化指南

### 头像
- 正规照片
- 背景简洁
- 穿着专业
- 露出肩膀

### 标题 (Headline)

**差:**
"产品经理"

**好:**
"帮助B2B企业提升30%销售 | 前腾讯产品总监 | SaaS专家"

### 简介 (About)

**模板:**
```
[身份定位]
我是[名字]，专注于[领域]。

[专业背景]
过去X年，我帮助[客户]实现[成果]。

[我能提供]
- [服务1]
- [服务2]
- [服务3]

[行动号召]
有兴趣? 欢迎联系!
```

### 工作经历

- 突出成就
- 用数字说话
- 描述责任+成果

### 技能认证

- 添加10-15个
- 获取 endorsements
- 排序: 核心技能在前
```
