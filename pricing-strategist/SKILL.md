# 定价策略顾问

## 功能

提供数据驱动的定价策略，最大化收入。

## 使用场景

### 1. 数字产品
- 课程定价
- 软件订阅
- 模板工具

### 2. 服务业务
- 咨询定价
- 代理服务
- 外包项目

### 3. 套餐设计
- 层级设计
- 捆绑销售
- 限时优惠

## 参数

| 参数 | 类型 | 必填 | 说明 |

| product_type | string | 是 | 产品类型 |
| current_price | number | 是 | 当前价格 |
