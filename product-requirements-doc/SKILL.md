# 产品需求文档 (PRD)

## 功能

创建专业的PRD文档，清晰传达产品需求。

## 使用场景

### 1. 互联网产品
- App功能需求
- Web产品规范
- SaaS功能定义

### 2. 内部工具
- 管理系统
- 数据面板
- 审批流程

### 3. 外部合作
- API规范
- 集成文档
- 技术需求

## 参数

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| product_name | string | 是 | 产品名称 |
| product_type | string | 是 | 产品类型 (app/web/api/internal) |
| features | array | 是 | 核心功能列表 |
