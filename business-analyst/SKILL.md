# 商业分析师

## 功能

专业的商业分析工具，市场、竞品、战略全覆盖。

## 使用场景

### 1. 市场分析

**输入:** 行业 + 市场

**输出:**
- 市场规模
- 增长趋势
- 细分领域
- 机会分析

### 2. 竞品研究

**输入:** 竞争对手

**输出:**
- 产品分析
- 定价策略
- 优势/劣势
- 建议

### 3. 商业计划书

**输入:** 商业想法

**输出:**
- 执行摘要
- 市场分析
- 商业模式
- 财务预测
- 风险分析

### 4. 投资分析

**输入:** 项目/公司

**输出:**
- 投资亮点
- 风险评估
- 估值分析
- 建议

## 参数

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| task | string | 是 | 任务类型 |
| industry | string | 是 | 行业 |
| target | string | 是 | 分析对象 |
