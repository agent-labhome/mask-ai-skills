# AI 自动化交易助手

## 功能

智能交易辅助工具，帮助你做交易决策，管理风险。

## 使用场景

### 1. 交易信号

**输入:** 交易对 + 时间框架

**输出:**
- 技术分析
- 关键支撑/阻力位
- 入场/出场建议
- 止盈止损建议

### 2. 仓位管理

**输入:** 账户余额 + 风险偏好

**输出:**
- 建议仓位大小
- 风险计算
- 分散投资建议

### 3. 止损策略

**输入:** 入场价 + 止损比例

**输出:**
- 动态止损建议
- 追踪止损方案
- 风险评估

### 4. 持仓分析

**输入:** 持仓列表

**输出:**
- 整体风险评估
- 建议调整
- 利润/亏损分析

## 参数

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| task | string | 是 | 任务类型 |
| symbol | string | 是 | 交易对 |
| timeframe | string | 否 | 时间框架 |
| balance | number | 否 | 账户余额 |
| risk | string | 否 | 风险偏好 |

## 示例

### 交易信号

**输入:**
```json
{
  "task": "信号",
  "symbol": "BTC/USDT",
  "timeframe": "4h"
}
```

**输出:**
```
📊 BTC/USDT 交易信号 (4h)

💰 当前价格: $66,427

🔍 技术分析:
- 趋势: 下跌 (MA20向下)
- RSI: 42 (超卖)
- MACD: 死叉

📍 关键位:
- 阻力: $68,000 / $70,000
- 支撑: $65,000 / $63,000

💡 操作建议:
- 做多: 突破$67,500入场
- 做空: 跌破$65,000入场
- 止损: ±3%

⚠️ 风险提示: 建议仓位 <20%
```

### 仓位管理

**输入:**
```json
{
  "task": "仓位",
  "balance": 10000,
  "risk": "稳健"
}
```

**输出:**
```
📊 仓位管理建议

💰 账户余额: $10,000
⚠️ 风险偏好: 稳健

📌 建议:
- 单笔最大仓位: $1,000 (10%)
- 建议持仓数: 3-5个
- 总仓位: <50%

🛡️ 风险管理:
- 总止损: <$500 (5%)
- 建议分散: 3个以上标的
```
