# Content Repurpose Engine

**触发词**: 内容再利用, 内容多平台分发, 内容变现

**功能描述**: 
将一个内容高效改造成多平台版本，最大化内容价值，包含视频、图文、音频等多种形式。

**使用场景**:
- 油管视频 → 短视频
- 博客 → Newsletter

- 课程 → 社交内容
