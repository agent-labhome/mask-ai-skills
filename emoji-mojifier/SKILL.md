# emoji-mojifier

将文字转换成emoji风格的文字表达。

## 使用场景

- 社交媒体发帖（Twitter/X、Instagram、Facebook）
- 消息发送时增加趣味性
- 内容创作时添加视觉元素
- 营销文案增强吸引力

## 功能

1. **文字转Emoji** - 将关键词替换为相关emoji
2. **心情转换** - 根据情感色彩添加适当表情
3. **文字装饰** - 添加装饰性emoji边框和分隔符
4. **批量处理** - 支持多段文字同时转换

## 使用方法

用户提供需要转换的文字，AI会：
1. 分析文字内容和情感
2. 识别可替换的关键词
3. 添加合适的emoji装饰
4. 输出转换后的文字

## 示例

输入: "今天天气真好，我们去公园野餐吧！"

输出: "☀️ 今天天气真好 🌤️，我们去公园野餐 🏕️ 吧！🎉"

## 输出格式

直接返回转换后的文字，无需额外说明。
