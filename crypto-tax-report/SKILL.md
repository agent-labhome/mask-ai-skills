# Crypto-Tax-Report
## 技能描述

**功能**: 加密货币税务报告生成器，帮你计算应税收入和申报数据

**使用场景**: 
- 年度税务申报
- 交易记录整理
- 成本基础计算

**触发词**: 加密税务,数字货币报税,crypto tax

**价格**: 免费

---

## 核心能力

1. **交易导入** - 支持CEX和DEX数据
2. **成本计算** - FIFO/LIFO/HIFO算法
3. **税务报告** - 生成各地区格式
4. **优化建议** - 税务筹划建议

---

## 使用方法

提供你的交易记录或导出文件，AI生成税务报告。

## 输出示例

```
📊 2025税务报告:
- 总交易量: 156笔
- 总收益: $45,230
- 应税收入: $32,150
- 预估税额: $8,039

📋 建议:
- 保留所有交易记录
- 考虑年度税务筹划
- 咨询专业税务顾问
```
