# Cat. Brief. - SKILL Documentation

## Overview
**Cat. Brief.** is a minimalist, high-speed note-taking application designed for capturing thoughts and ideas with maximum efficiency. The application prioritizes speed and simplicity over complex features, making it ideal for quick note capture without distractions.

## Core Philosophy
- **Speed First**: Open, type, save - in under 3 seconds
- **Minimalist Design**: No toolbars, no formatting options, just pure text
- **Keyboard-Driven**: Full operation without touching the mouse
- **Zero Distraction**: No auto-save animations, no confirmation dialogs

## Technical Stack
- **Frontend**: HTML, CSS, JavaScript (Vanilla)
- **Storage**: LocalStorage for immediate persistence
- **Design**: Custom minimalist interface with system font stack
- **No Dependencies**: Pure vanilla implementation
