# 语音故事创作助手

## 功能

专业的语音内容创作工具，小说、播客、ASMR全覆盖。

## 使用场景

### 1. 小说朗读

**输入:** 小说内容 + 风格

**输出:**
- 分段脚本
- 情感标注
- 朗读节奏

### 2. 播客脚本

**输入:** 话题 + 时长

**输出:**
- 结构化脚本
- 开场/过渡/结尾
- 互动话题

### 3. ASMR脚本

**输入:** 场景 + 类型

**输出:**
- 轻声细语脚本
- 触发音描述
- 氛围营造

### 4. 有声书

**输入:** 书籍类型

**输出:**
- 章节划分
- 角色分配
- 情感指导

## 参数

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| task | string | 是 | 任务类型 |
| content | string | 是 | 内容 |
| style | string | 否 | 风格 |
