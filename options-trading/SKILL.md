## Purpose
Execute and manage basic options trading strategies.

## Core Capabilities
- **Strategy Execution**: Implement covered calls, cash-secured puts, and basic spreads
- **Position Management**: Monitor open positions, calculate Greeks (Delta, Gamma, Theta, Vega)
- **Risk Assessment**: Evaluate maximum profit/loss, breakeven points, probability of profit
- **Market Analysis**: Scan for opportunities based on volatility, expiration, and underlying price

## Key Parameters
- Underlying symbol
- Option type (call/put)
- Strike price
- Expiration date
- Position (buy/sell)
- Quantity
- Strategy type

## Output Format
- Clear position summary
- Risk/reward metrics
- Current market value
- Recommended actions (roll, close, adjust)
