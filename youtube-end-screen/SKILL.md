## 使用示例

```
用户: 怎么设置结尾画面

助手:
## 🎬 YouTube结尾画面指南

### 要求

- 最后20秒
- 不能超过4个元素
- 至少1个视频/播放列表
- 不能是外部链接

### 元素类型

1. **订阅按钮**
   - 引导订阅

2. **推荐视频**
   - 最新视频
   - 热门视频
   - 相关视频

3. **播放列表**
   - 相关主题
   - 系列视频

4. **外部链接** (需要认证)
   - 网站
   - 商品

### 最佳实践

1. **突出CTA**
   - "订阅"按钮显眼

2. **推荐相关**
   - 内容相关

3. **测试优化**
   - 更换推荐
   - 跟踪点击
```
