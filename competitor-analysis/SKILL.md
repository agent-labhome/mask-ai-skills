# 竞品分析报告

## 功能

生成专业的竞品分析报告，辅助商业决策。

## 使用场景

### 1. 产品定位
- 市场份额
- 差异化分析
- 定价策略

### 2. 功能对比
- 功能矩阵
- 用户体验
- 技术架构

### 3. 营销分析
- 推广渠道
- 内容策略
- 用户评价

## 参数

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| competitors | array | 是 | 竞品列表 |
| analysis_type | string | 是 | 分析类型 (product/marketing/Full) |
| industry | string | 是 | 行业 |
