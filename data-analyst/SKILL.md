## 示例

### 数据分析

**输入:**
```json
{
  "task": "分析",
  "data": "月销售数据",
  "goal": "找出销售趋势"
}
```

**输出:**
```
📊 数据分析报告

📈 关键发现:
1. 月度销售呈上升趋势 (+15% MoM)
2. 周末销售明显高于工作日
3. 畅销产品: 电子产品 (占比45%)
4. 客户来源: 线上70% / 线下30%

📉 异常:
- 3月销量异常低 (可能是季节因素)

💡 建议:
1. 加大周末促销力度
2. 重点推广电子品类
3. 增加线上渠道投入
```
