# Course Creator - 在线课程创建助手

帮助你从零开始创建、发布和销售在线课程。

## 功能
- 自动执行任务
- 提供专业建议
- 生成优化方案

## 课程创建流程

### 1. 定位与选题

确定课程方向：
- 目标受众是谁？解决什么痛点？
- 课程形式：视频、音频、图文、直播？
- 课时数量：5-30节课
- 定价区间：免费/付费($19-$999)

热门领域：
- 编程开发、AI工具使用
- 商业创业、副业赚钱
- 个人成长、效率管理
- 设计创意、营销推广
- 语言学习、职业技能

### 2. 课程大纲设计

每门课程应包含：

**引流模块（免费）**
- 简介视频（3-5分钟）
- 课程大纲PDF
- 免费试听课

**核心内容模块**
- 每个模块3-8节课
- 每节课10-30分钟
- 理论+实操结合

**增值服务（可选）**
- 直播答疑
- 作业批改
- 社群支持
- 1对1咨询

### 3. 内容创作要点

**视频脚本结构**：
```
[开场] 1分钟 - 介绍本节目标
[导入] 2分钟 - 关联上节/引入问题
[讲解] 5-15分钟 - 核心内容
[演示] 3-5分钟 - 实操演示
[总结] 1分钟 - 关键要点回顾
[行动] 30秒 - 下节预告/作业
```

**脚本技巧**：
- 每3分钟换一个画面/场景
- 关键信息重复3遍
- 使用故事和案例
- 保持语速适中

### 4. 定价策略

定价因素：
- 内容深度和完整度
- 附加值（社群、答疑、证书）
- 目标受众支付能力
- 市场竞争情况

参考定价：
- 入门课程：$19-$49
- 进阶课程：$49-$199
- 高级课程：$199-$499
- 企业培训：$499-$999+

### 5. 发布平台选择

| 平台 | 特点 | 适合类型 |
|------|------|----------|
| Udemy | 流量大、抽成高 | 大众课程 |
| Teachable | 专业品牌 | 独立讲师 |
| Kajabi | 一站式解决方案 | 高端课程 |
| Gumroad | 简单快速 | 数字产品 |
| 自建站 | 完全控制 | 长期运营 |

### 6. 营销推广

**Launch策略**：
- 预告期：2-4周
- 预售期：1-2周（早鸟价）
- 正式发布：限时优惠
- 后续：常规销售

**引流渠道**：
- YouTube预告片
- Twitter/小红书种草
- 播客访谈
- 联盟营销
- 邮件列表

### 7. 学员支持

- Discord/Telegram社群
- 每周直播答疑
- 作业提交与反馈
- 毕业证书
- 学员成功案例征集

## 常用命令

创建课程文件夹结构：
```bash
mkdir -p course-name/{scripts,videos,assets,references}
```

## 资源推荐

- 视频录制：OBS、Camtasia、ScreenFlow
- 剪辑：Final Cut Pro、Premiere、DaVinci Resolve
- 音频：Audacity、Adobe Podcast
- 托管：Teachable、Kajabi、Thinkific
- 支付：Stripe、Gumroad、Paddle
