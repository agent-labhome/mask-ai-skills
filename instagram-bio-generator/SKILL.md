# Instagram Bio 生成器

## 核心原则

1. **简洁有力** - Bio最多150字符，必须一针见血
2. **价值明确** - 让粉丝一眼知道能获得什么
3. **行动导向** - 包含CTA引导互动或关注

## Bio类型模板

### 个人品牌型
```
🎯 [核心身份/职业]
✨ [你能提供什么]
📍 [可选位置]
👉 [CTA]
```

### 博主/内容创作者型
```
[身份标签] | [内容领域]
📝 [代表作/成就]
🔔 [更新频率]
👇 [热门内容链接]
```

### 品牌/商家型
```
[品牌名] - [slogan]
🌟 [产品/服务亮点]
⭐️ [社会证明]
🛒 [购买链接]
```

### 网红/影响者型
```
[昵称] | [垂直领域]
📸 [内容风格]
📈 [粉丝数/数据]
💌 [合作邮箱]
```

## Emoji使用指南

- 定位标签: 🎯 📍 🌍
- 行动号召: 👉 👇 🔗
- 成就展示: ⭐️ 🏆 ✅
- 内容类型: 📸 🎬 📝 🎙️
- 品牌元素: ✨ 🌟 💫 🔥

## CTA最佳实践

| 目标 | CTA示例 |
|------|---------|
| 关注 | "点个关注不迷路" / " Follow for more" |
| 点击 | "链接在主页" / "Link in bio" |
| 互动 | "评论区见" / "DM for collab" |
| 购物 | "🛒 Shop now" / "Get 20% off" |

## 生成技巧

1. **定位清晰**: 先确定账号定位和目标受众
2. **差异化**: 突出独特卖点，避免同质化
3. **关键词**: 包含受众会搜索的关键词
4. **测试优化**: A/B测试不同版本，持续优化
