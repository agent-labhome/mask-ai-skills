# linkedin-outreach

LinkedIn开发助手 - 个性化消息、连接请求、内容策略

## 描述
帮助用户通过LinkedIn进行有效的网络拓展和潜在客户开发。包括个性化连接消息、跟进策略、内容发布建议。

## 使用场景
- cold outreach (冷联系)
- 招聘拓展
- B2B销售线索
- 行业人脉建立
- 品牌建设

## 核心功能
1. 个性化连接消息生成
2. 冷消息模板
3. 跟进策略制定
4. 内容策略建议
5. InMail优化

## 提示词模板

### 连接请求
```
帮我写一个LinkedIn连接请求：
- 目标受众：[描述]
- 目的：[建立人脉/招聘/合作]
- 背景：[如何认识对方或共同点]
```

### 冷消息
```
生成冷消息：
- 行业：[行业名称]
- 痛点：[目标客户痛点]
- 解决方案：[你的产品/服务]
- 行动号召：[想要对方做什么]
```

### 跟进消息
```
跟进消息模板：
- 之前的消息：[之前发送的内容]
- 跟进目的：[提醒/追加价值]
- 时间间隔：[几天后跟进]
```

## 最佳实践
- 保持简短（不超过300字符）
- 提及共同点或共同人脉
- 提供价值而非直接推销
- 个性化每个消息
- 选择合适的发送时间
