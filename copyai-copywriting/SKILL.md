# Copy.ai AI文案写作

## 功能

利用Copy.ai生成各种营销文案，包括社交媒体、邮件、广告、产品描述等。

## 使用场景

### 1. 社交媒体
- Instagram caption
- Twitter推文
- LinkedIn帖子

### 2. 营销内容
- 广告文案
- Landing Page文案
- 邮件主题行

### 3. 产品文案
- 产品描述
- 功能介绍
- 客户评价

## 参数

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| content_type | string | 是 | 内容类型 |
| tone | string | 是 | 文案风格 |
| target_audience | string | 是 | 目标受众 |
| word_count | number | 是 | 字数要求 |
