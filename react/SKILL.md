## Core Concepts
- **Components**: Function/class-based reusable UI pieces
- **JSX**: JavaScript XML syntax for UI structure
- **State**: Component-specific data that triggers re-renders when changed
- **Props**: Read-only data passed from parent to child components
- **Hooks**: Functions that let you use React features in functional components

## Essential Hooks
- `useState`: Manage component state
- `useEffect`: Handle side effects (API calls, subscriptions)
- `useContext`: Access React context
- `useRef`: Reference DOM elements or persist values across renders
