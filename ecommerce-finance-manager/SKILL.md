# Ecommerce Finance Manager

## 基础信息
- **Skill名称**: 电商财务管理助手
- **目录**: ecommerce-finance-manager
- **触发词**: 电商财务、账务管理、利润计算、税务合规、成本分析
- **价格**: $39/$89/$199

## 功能描述

### 核心功能
1. **利润计算** - 计算单品/全店利润，识别高毛利产品
2. **成本分析** - 分析物流、平台费用、广告成本占比
3. **税务合规** - 提供跨境电商税务基础知识
4. **现金流管理** - 监控应收账款、库存占用资金
5. **财务报告生成** - 生成月度/季度财务分析报告

### 使用场景
- 跨境电商店长需要了解真实利润
- 财务人员需要自动化报表
- 创业者需要成本控制建议

## 用户痛点
- 不知道产品真正赚了多少钱
- 平台费用复杂难以计算
- 税务问题不知如何处理

## 变现路径
- 提供财务模板 → 付费模板
- 咨询服务 → 高客单价
- 培训课程 → 持续收入
