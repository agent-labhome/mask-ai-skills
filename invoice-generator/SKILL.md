# Invoice-Generator
## 技能描述

**功能**: 跨境电商/自由职业发票生成器，支持多币种和格式

**使用场景**: 
- 跨境收款发票
- 自由职业报价单
- 合同附件

**触发词**: 发票生成,invoice,报价单

**价格**: 免费

---

## 核心能力

1. **多币种** - USD/EUR/GBP/CNY/HKD
2. **合规模板** - 符合各国要求
3. **自动计算** - 税费和折扣
4. **PDF导出** - 专业格式

---

## 输入格式

```
客户:
服务内容:
金额:
币种:
```
