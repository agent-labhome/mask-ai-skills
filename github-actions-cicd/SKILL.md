# Github Actions Cicd

## 触发词
github actions, ci cd, 自动化部署, 工作流

## 功能描述
GitHub Actions是GitHub的CI/CD平台，支持自动化构建、测试和部署工作流。

## 使用场景
- 持续集成
- 自动化测试
- 持续部署
- 定时任务
- 自动化工作流
