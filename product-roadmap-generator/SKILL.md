# Product Roadmap Generator

_AI-powered product roadmap creation and planning assistant_

## When to Use

- 创建产品路线图
- 规划功能优先级
- 制定发布计划
- 产品战略规划
- 迭代规划

## What It Does

帮助用户创建专业的产品路线图，包括：
- 阶段规划
- 优先级排序
- 时间线规划
- 里程碑设定
- 资源分配建议

## How to Use

用户提供：
- 产品名称和描述
- 目标用户群体
- 核心功能想法
- 预期发布时间

输出：
- 结构化路线图
- 优先级矩阵
- 发布计划时间表
- 建议和最佳实践

## Prompt Template

```
请帮我创建[产品名称]的产品路线图。

产品描述：[描述]
目标用户：[用户群体]
核心功能想法：
1. [功能1]
2. [功能2]
3. [功能3]

预期发布时间：[时间]

请提供：
- 阶段规划
- 功能优先级
- 发布计划
- 里程碑
```
