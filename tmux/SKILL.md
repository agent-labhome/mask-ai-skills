# Tmux Controller

## 功能

远程控制tmux会话，自动化CLI操作。

## 使用场景

### 1. 远程操作

**输入:** 命令

**输出:**
- 执行结果
- 输出内容
- 状态

### 2. 会话管理

**输入:** 操作

**输出:**
- 会话列表
- 创建/删除
- 切换

### 3. 多窗口操作

**输入:** 窗口操作

**输出:**
- 窗口管理
- 布局调整
- 同步输入

### 4. 自动化脚本

**输入:** 任务

**输出:**
- 脚本生成
- 定时执行
- 监控

## 示例

### 远程执行

**输入:**
```json
{
  "task": "执行",
  "command": "ls -la"
}
```

**输出:**
```
✅ 执行成功

total 64
drwxr-xr-x  5 user  user 4096 Feb 19 11:00
drwxr-xr-x  3 user  user 4096 Feb 19 10:30
...
```
