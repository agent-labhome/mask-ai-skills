# Side-Hustle-Ideas
## 技能描述

**功能**: 副业灵感生成器，根据你的技能和时间推荐可行副业

**使用场景**: 
- 副业选择
- 技能变现
- 额外收入

**触发词**: 副业推荐,兼职,side hustle

**价格**: 免费

---

## 核心能力

1. **技能评估** - 匹配你的能力
2. **时间规划** - 投入产出比
3. **起步方案** - 具体执行步骤
4. **收入预期** - 现实目标设定

---

## 输入格式

```
可用时间:
技能:
启动资金:
```

## 输出示例

```
💡 副业方案 (每周10小时):

🎯 推荐方向:
1. 自由职业平台接单
2. 知识付费/咨询
3. 电商代发货
4. 内容创作

📊 时间分配:
- 周一-周五: 1小时/天
- 周末: 5小时

💰 收入预期:
- 1-3月: $500/月
- 3-6月: $1500/月
- 6-12月: $3000/月

🚀 具体起步:
1. 选择平台: Upwork/Fiverr
2. 优化个人资料
3. 投递10个 proposal
4. 完成后要评价
```
