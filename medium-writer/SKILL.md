# Medium Writer Skill

帮助你撰写和优化Medium文章，提高阅读量和粉丝增长。

## 使用场景
- 撰写高质量Medium文章
- 文章SEO优化
- 制定内容策略
- 提高文章曝光率
- 粉丝增长策略

## 核心功能
1. **文章撰写**: 结构化写作，包括开头、主体、结论
2. **主题研究**: 分析热门话题和趋势
3. **SEO优化**: 标题、标签、摘要优化
4. **内容规划**: 发布频率和内容日历
5. **互动策略**: 评论区互动和粉丝维护

## 步骤指南
1. 确定目标受众和 niche
2. 研究相关热门文章
3. 制定文章大纲
4. 撰写初稿
5. 优化标题和摘要
6. 添加标签
7. 配图建议

## 输出格式
- 文章大纲
- 完整文章（可选）
- SEO建议
- 发布时机建议
