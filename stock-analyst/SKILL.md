# 股票分析助手

## 功能

专业的股票分析工具，基本面、技术面、组合管理。

## 使用场景

### 1. 基本面分析

**输入:** 股票代码

**输出:**
- 财务指标
- 估值分析
- 投资建议

### 2. 技术分析

**输入:** 股票 + 周期

**输出:**
- 趋势判断
- 支撑阻力
- 信号提示

### 3. 组合管理

**输入:** 持仓

**输出:**
- 风险评估
- 仓位建议
- 调仓策略

### 4. 财报解读

**输入:** 财报发布

**输出:**
- 关键数据
- 业绩分析
- 预期调整

## 参数

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| task | string | 是 | 任务类型 |
| stock | string | 是 | 股票代码 |
| market | string | 是 | 市场 |
