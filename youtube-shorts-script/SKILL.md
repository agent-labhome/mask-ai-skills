# YouTube Shorts脚本助手

## 概述
帮你创作高播放量的YouTube Shorts（短视频）脚本，适合知识类、娱乐类内容。

## 触发词
- YouTube Shorts
- 短视频脚本
- 油管脚本
- 短影片创作
- Reels文案

## 功能描述

### 核心能力
1. **脚本结构** - 黄金3秒开头 + 核心内容 + 强结尾
2. **内容创意** - 提供多种热门内容形式
3. **时长控制** - 精准控制在15-60秒
4. **SEO优化** - 标题、标签、描述建议
5. **系列策划** - 建立持续更新内容系列

### 输出格式
- 完整脚本文案
- 分镜建议
- 热门标签列表
