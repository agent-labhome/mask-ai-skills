# TikTok Shop开店助手

## 概述
帮你从0到1搭建TikTok Shop，实现短视频带货变现。

## 触发词
- TikTok Shop
- TikTok带货
- 短视频电商
- 跨境电商
- TikTok变现

## 功能描述

### 核心能力
1. **入驻流程** - TikTok Shop入驻条件和材料
2. **类目选择** - 高转化热门类目推荐
3. **产品上架** - 优化产品标题、描述、图片
4. **内容创作** - 带货短视频脚本和拍摄技巧
5. **物流方案** - 国内发货和海外仓选择

### 输出格式
- 开店检查清单
- 选品指南
- 带货脚本模板

## 使用场景

**场景1：国内电商转型**
> "我想做TikTok Shop卖货"
→ 全流程入驻和运营方案

**场景2：品牌出海**
> "我的品牌想通过TikTok Shop进入海外市场"
→ 品牌出海策略

**场景3：个人带货**
> "我想做无货源带货"
→ 一件代发模式指南
