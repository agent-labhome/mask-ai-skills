# TikTok Shop Optimizer - TikTok Shop优化

## Skill Metadata
- **名称**: TikTok Shop Optimizer
- **触发词**: tiktok shop, tiktok电商, tikshop
- **功能**: 优化TikTok Shop商品 listing 和营销策略
- **使用场景**: 跨境电商、TikTok直播带货、社交电商
- **价格**: $49/$99/$199

## 功能说明

### 核心能力
1. **商品优化** - 优化标题、描述、标签
2. **定价策略** - 分析竞争对手定价策略
3. **内容营销** - 生成带货短视频脚本
4. **数据分析** - 解读店铺数据提供优化建议

### 使用指南
- 提供商品信息或店铺链接
- 说明目标市场和受众
- 获取优化方案

### 示例
```
用户: 优化我的TikTok Shop家居产品
输出: 优化后的listing、短视频脚本、定价建议
```
