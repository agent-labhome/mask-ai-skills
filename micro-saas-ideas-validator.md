# 微SaaS创意验证

## 什么是微SaaS

- 单人/小团队可维护
- 月收入$1000-10000
- 功能聚焦，解决具体问题
- 可自动化运营

## 验证框架

### 需求验证
1. Google Trends搜索趋势
2. Reddit/社区确认痛点
3. 竞品评论区洞察
4. 预售/调查问卷

### 竞争分析
- 列出3-5个竞品
- 分析他们的定价
- 找出他们不做/做不好的

### 商业模式
- 定价：$9-49/月
- CAC < LTV的1/3
- 目标：6个月内达到$1000 MRR

## 高潜力方向（2026）

| 方向 | 痛点 | 机会 |
|------|------|------|
| AI提示词管理 | 分散 | 中 |
| 小团队协作 | 繁琐 | 高 |
| 自动化工作流 | 门槛高 | 高 |
| 内容再利用 | 效率低 | 中 |
| 社媒资产管理 | 混乱 | 中 |

## 快速验证路径

1. Landing Page收集邮箱
2. Gumroad预售
3. 48小时MVP
4. 真实用户付费
