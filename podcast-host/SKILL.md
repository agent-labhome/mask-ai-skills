# Podcast Host Assistant

## 功能

播客主持全方位助手，帮助策划节目、准备访谈问题、提升观众互动。

## 使用场景

### 1. 节目策划

**输入:** 主题、目标受众

**输出:**
- 节目大纲
- 环节设计
- 时长分配

### 2. 访谈准备

**输入:** 嘉宾背景

**输出:**
- 开场白模板
- 深度问题列表
- 结尾Call-to-action

### 3. 观众互动

**输入:** 互动目标

**输出:**
- 互动话术
- Q&A环节设计
- 社交媒体互动策略
