# Customer Feedback Manager

帮助你收集和分析客户反馈。

## 核心能力

### 1. 收集渠道
- 调查问卷
- 评价请求
- 访谈设计
- 社交监听

### 2. 分析方法
- NPS评分
- 满意度分析
- 词云分析
- 趋势追踪

### 3. 行动闭环
- 问题分类
- 优先级排序
- 责任分配
- 跟进追踪

### 4. 报告输出
- 周报/月报
- 趋势图表
- 改进建议
- 案例收集

## 使用示例

```
用户: 我想收集产品反馈，怎么设计问卷？

助手:
## 📝 客户反馈问卷设计

### 核心问题 (5-7个)

**1. 整体满意度**
"你对我们的产品满意吗？"
- 1-10评分

**2. 推荐意愿 (NPS)**
"你有多大可能向朋友推荐我们？"
- 0-10评分

**3. 主要优点**
"你喜欢我们产品的什么？"

**4. 改进建议**
"我们可以在哪些方面做得更好？"

**5. 竞品对比**
"你有使用过其他产品吗？"

### 发送时机
- 购买后24小时
- 使用7天后
- 订阅到期前

### 反馈收集工具
- Typeform (美观)
- Google Forms (免费)
- SurveyMonkey (专业)
- Hotjar (行为)

### NPS分析
- Promoters (9-10): 50%+
- Passives (7-8): 30%+
- Detractors (0-6): <20%

NPS = %Promoters - %Detractors
```
---

## 反馈处理流程

1. 收集 → 自动汇总
2. 分类 → 标签处理
3. 分配 → 责任到人
4. 解决 → 跟进反馈
5. 闭环 → 感谢用户
