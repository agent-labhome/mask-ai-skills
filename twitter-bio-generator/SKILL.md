# Twitter Bio 生成器

## 核心原则

1. **简洁精炼** - Bio最多160字符，必须字字珠玑
2. **身份明确** - 让粉丝一眼知道你是谁、做什么
3. **个性化** - 体现独特风格和声音
4. **行动导向** - 包含CTA引导关注或互动

## Bio类型模板

### 个人品牌型
```
[身份/职业] | [核心标签]
[你能提供什么]
[社交证明/成就]
👇 [CTA]
```

### 博主/内容创作者型
```
[内容领域] Creator
[代表作/成就]
[更新频率]
🔔 [CTA]
```

### 开发者/技术型
```
[职位] @[公司]
[技术栈/专长]
[开源项目/产品]
[联系方式]
```

### 创业者/商业型
```
[角色] | [垂直领域]
[公司/产品]
[成就/数据]
[合作联系]
```

### 网红/影响者型
```
[昵称] | [垂直领域]
[内容风格]
[粉丝数/数据]
📩 [合作邮箱]
```

## 生成技巧

1. **定位清晰**: 确定账号定位和目标受众
2. **差异化**: 突出独特卖点，避免同质化
3. **关键词**: 包含受众会搜索的关键词
4. **品牌一致**: 与内容风格保持一致
5. **测试优化**: 持续测试不同版本
