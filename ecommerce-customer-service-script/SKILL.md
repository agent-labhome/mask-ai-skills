# Ecommerce Customer Service Script

## 描述
为电商卖家提供专业客服话术，提高客户满意度和服务效率。

## 触发词
- 电商客服
- 客服话术
- 客户沟通
- 售后处理
- 差评处理
- 咨询回复

## 功能

### 1. 售前话术
- 产品咨询
- 价格议价
- 物流咨询
- 促销咨询

### 2. 售中话术
- 订单确认
- 物流更新
- 异常处理
- 催付款

### 3. 售后话术
- 退换货处理
- 差评挽回
- 投诉处理
- 好评引导

### 4. 情绪处理
- 愤怒客户
- 急躁客户
- 犹豫客户
- 纠缠客户

## 使用场景

**场景1: 客服培训**
用户: "新招客服不知道怎么回复"
模板: 各类场景话术+培训指南

**场景2: 差评处理**
用户: "收到一个差评怎么办"
方案: 差评挽回策略+话术

**场景3: 效率提升**
用户: "客服回复太慢"
方案: 话术模板+快捷回复
