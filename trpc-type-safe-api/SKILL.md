# Trpc Type Safe Api

## 触发词
trpc, 类型安全, type-safe API, 前后端类型

## 功能描述
tRPC是无需编写API schema的类型安全API框架，实现前后端类型共享。

## 使用场景
- 全栈TypeScript项目
- Next.js应用
- 微服务架构
- 内部工具开发
