# LinkedIn B2B开发信

## 为什么LinkedIn B2B

- 9亿+专业用户
- 决策者直接触达
- 可验证公司/职位
- 竞争比邮件低

## 开发信模板

### 价值型开场
```
Hi [Name],

I noticed [具体观察] and thought you'd be interested in [价值点].

[简短案例/数据]

Interested in a 15-min call?

Best,
[Your Name]
```

### 资源型开场
```
Hi [Name],

We helped [类似公司] achieve [具体结果].

Thought you might find our [资源] useful for [痛点].

[链接/附件]

Best,
[Your Name]
```

## 规模化策略

1. **Sales Navigator** - 高级搜索
2. **自动化工具** - Expandi, Dux-Soup
3. **内容沉淀** - 持续价值输出
4. **裂变机制** - 推荐奖励

## 常见拒绝话术

- "太忙" → 提供异步方案
- "预算不足" → 讨论范围
- "不需要" → 保持联系
- "已有供应商" → 差异化
