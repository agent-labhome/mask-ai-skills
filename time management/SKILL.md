# Time Management

## 功能
- 自动执行任务
- 提供专业建议

## Core Principles
- **Prioritize** tasks using the Eisenhower Matrix (Urgent/Important)
- **Plan** your day with time blocking
- **Focus** using techniques like Pomodoro (25-minute work sprints)
- **Limit** distractions and multitasking
- **Review** daily to improve tomorrow

## Quick Actions
- Start each day identifying your **Top 3** tasks
- Use **time blocking** to schedule deep work
- **Batch** similar tasks together
- **Delegate** or eliminate low-value activities
- **Say no** to protect your priorities

## Key Tools
- Calendar for appointments
- Task list for priorities
- Timer for focused work sessions
- "Do Not Disturb" for concentration
