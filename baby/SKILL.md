## Overview
Baby.Brief is a minimalist, privacy-focused URL shortener that transforms long URLs into brief, memorable links. Built with simplicity and user experience at its core, it provides essential URL shortening functionality without unnecessary complexity.

## Core Features

### 1. **URL Shortening**
- **Function**: `shorten(url, custom_alias?)`
- **Input**: Long URL (required), custom alias (optional)
- **Output**: Shortened URL with unique code
- **Example**: 
  ```
  Input: https://www.example.com/very-long-path/to/some/resource
  Output: https://baby.brief/abc123
  ```
