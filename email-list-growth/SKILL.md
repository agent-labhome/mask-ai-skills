## 使用示例

```
用户: 我想增加邮件订阅用户

助手:
## 📧 邮件列表增长指南

### Lead Magnet类型
1. **PDF指南** - "X个技巧"
2. **清单** - "X个资源"
3. **模板** - "现成模板"

- 触发: 退出意图/时间/滚动

- 设计: 简洁+明确CTA

### 落地页模板
```
标题: 免费获得[价值]
副标题: 加入X人订阅行列
表单: 邮箱 + (名字)
CTA: 免费获取
社会证明: "X人已订阅"
```

### 增长技巧
- 博客内嵌注册
- 社交媒体推广
- 播客介绍
- 合作互推
- Webinary

### 发送频率
- 初期: 1封/周
- 增长: 2封/周
- 稳定: 按内容需求

### 清理策略
- 60天不活跃 → 重新激活序列
- 不打开 → 减少频率
- 退订 → 尊重选择
```
