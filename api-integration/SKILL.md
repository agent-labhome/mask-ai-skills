## Core Competencies
- RESTful API design and consumption
- Authentication methods (API keys, OAuth, JWT)
- HTTP methods, status codes, and error handling
- JSON/XML data parsing and serialization
- Rate limiting and pagination strategies
- Webhook implementation and management

## Key Tools
- **Testing**: Postman, Insomnia, curl
- **Documentation**: OpenAPI/Swagger, API Blueprint
- **Libraries**: Axios, Requests, Fetch API
- **Monitoring**: API analytics, logging, uptime tracking

## Best Practices
- Implement robust error handling and retry logic
- Use environment variables for sensitive credentials
- Cache responses when appropriate
- Validate request/response schemas
- Write comprehensive integration tests
- Document integration patterns and edge cases
