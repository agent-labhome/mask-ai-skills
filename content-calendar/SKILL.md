# Content Calendar Manager

帮助你规划和管理内容日历。

## 核心能力

### 1. 内容规划
- 月度计划
- 周度安排
- 主题确定
- 节奏把控

### 2. 平台适配
- Twitter
- LinkedIn
- Instagram
- YouTube

### 3. 工具选择
- Notion
- Airtable
- Google Sheets
- 专用工具

### 4. 执行追踪
- 发布状态
- 数据追踪
- 复盘分析
- 优化迭代

## 使用示例

```
用户: 我想规划下个月的内容

助手:
## 📅 内容日历规划

### 月度主题

**1月主题:** "AI创业指南"

**Week 1: 认知**
- 什么是AI创业
- 机会在哪里
- 需要什么技能

**Week 2: 起步**
- 如何找方向
- MVP怎么写
- 第一批用户

**Week 3: 增长**
- 获客策略
- 产品迭代
- 团队建设

**Week 4: 变现**
- 商业模式
- 定价策略
- 案例分析

### 每日内容

| 日期 | 平台 | 内容 |
|------|------|------|
| 周一 | Twitter | 干货thread |
| 周二 | LinkedIn | 深度文章 |
| 周三 | Instagram | 短视频 |
| 周四 | YouTube | 教程视频 |
| 周五 | 社群 | 互动话题 |

### 工具模板

**Notion模板:**
- 视图: Calendar / List / Board

- 筛选: 按平台/状态
```
