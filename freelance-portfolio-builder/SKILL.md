# 自由职业作品集助手

## 概述
帮自由职业者打造专业作品集网站，提升接单成功率。

## 触发词
- 自由职业
- 作品集网站
- Portfolio
- 接单网站
- 个人官网

## 功能描述

### 核心能力
1. **作品集结构** - 展示核心能力和代表性案例
2. **案例呈现** - 用故事化方式展示项目成果
3. **自我介绍** - 有说服力的个人介绍
4. **转化设计** - 引导潜在客户联系你
5. **技术方案** - Carrd、Notion、Webflow等工具推荐

### 输出格式
- 作品集结构大纲
- 文案模板
- 工具配置指南

## 使用场景

**场景1：设计师/创意类**
> "我是平面设计师，想展示作品吸引客户"
→ 视觉导向作品集

**场景2：开发者/技术类**
> "我是程序员，想接外包项目"
→ 技术项目展示

**场景3：咨询/服务类**
> "我是营销顾问，想展示案例"
→ 成果导向作品集
