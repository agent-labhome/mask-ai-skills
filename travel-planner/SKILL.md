# 旅行规划助手

## 功能

智能旅行规划工具，行程、预算、景点全覆盖。

## 使用场景

### 1. 行程安排

**输入:** 目的地 + 天数 + 预算

**输出:**
- 每日行程
- 交通建议
- 时间安排
- 备用计划

### 2. 预算计算

**输入:** 目的地 + 天数

**输出:**
- 预算明细
- 节省建议
- 支付方式

### 3. 景点推荐

**输入:** 目的地 + 偏好

**输出:**
- 必去景点
- 小众秘境
- 美食推荐

### 4. 行前准备

**输入:** 目的地

**输出:**
- 签证要求
- 天气穿着
- 必备物品
- 安全注意

## 参数

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| task | string | 是 | 任务类型 |
| destination | string | 是 | 目的地 |
| days | number | 是 | 天数 |
