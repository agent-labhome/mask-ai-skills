# SKILL.md for B2B SaaS Projects

## Project Context
This document outlines the skills, tools, and approaches needed for successful B2B SaaS development. B2B SaaS differs significantly from consumer products in complexity, sales cycles, compliance requirements, and user expectations.

## Core Technical Stack

### Backend & Infrastructure
- **Languages/Frameworks**: Python (Django/Flask/FastAPI), Java (Spring), Node.js, Go, .NET Core
- **Database**: PostgreSQL (preferred for ACID compliance), MySQL, MongoDB (for specific use cases), Redis (caching)
- **Cloud Infrastructure**: AWS (most common), Azure (enterprise preference), GCP
- **Containerization**: Docker, Kubernetes (for scaling)
- **API Design**: RESTful APIs, GraphQL (for complex data relationships), gRPC (for microservices)
- **Message Queues**: RabbitMQ, Kafka, AWS SQS
