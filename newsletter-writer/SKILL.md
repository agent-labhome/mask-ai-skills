# Newsletter写作助手 - 内容策划、写作、发布

## 描述
专为Newsletter创作者设计，帮你策划内容、撰写高质量文章、管理订阅用户和优化开信率。

## 使用场景
- 每周/每日Newsletter更新
- 垂直领域内容策展
- 邮件营销序列
- 个人品牌 Newsletter

## 核心功能
1. **内容策划** - 根据主题和受众推荐内容方向
2. **标题优化** - 生成多个吸引人的标题选项
3. **文章撰写** - 结构化写作，包括开头、主体、结尾
4. **排版优化** - Markdown格式，适合邮件阅读
5. **CTA设计** - 行动号召文案优化
6. **开信率优化** - 预览文本、发送时间建议

## 输入格式
```
主题: [你的Newsletter主题]
当前订阅数: [人数]
目标受众: [描述]
本周重点: [1-3个话题]
```

## 输出示例
-  Newsletter结构大纲
- 3-5个可选标题
- 完整文章草稿
- CTA建议
- 最佳发送时间
