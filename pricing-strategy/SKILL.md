# Pricing Strategy

定价是收入的核心。本技能帮助制定最优定价策略。

## 定价心理

### 数字心理学
- $99 > $100 (尾数效应)
- $49.99 > $50 (折扣感知)
- $997 > $999 (整数结尾 = 高端)

### 价格锚点
- 展示原价 vs 优惠价
- 高价选项衬托中间选项
- 示例："专业版 $199/月，企业版 $499/月"

## 定价模型

### 1. 成本加成定价
```
价格 = 成本 × (1 + 利润率)
```
适合：实体产品

### 2. 价值定价
```
价格 = 客户获得的价值 × 价值感知系数
```
适合：SaaS、在线课程

### 3. 竞争定价
```
价格 = 竞争对手价格 ± 差异化溢价
```
适合：竞争激烈市场

### 4. 渗透定价
```
低价进入 → 积累用户 → 逐步涨价
```
适合：需要市场份额的产品

### 5. 订阅定价
```
月度 = 年付 ÷ 12 × 1.2
年度 = 月付 × 10
```
常用区间：$9/月, $29/月, $99/月, $299/月

## 产品线定价

### 经典三档

| 层级 | 价格 | 定位 |
|------|------|------|
| 入门 | $19-$49 | 降低门槛 |
| 专业 | $99-$199 | 主推产品 |
| 企业 | $499+ | 最高价值 |

### 价格差距
- 入门到专业：2-3倍
- 专业到企业：3-5倍
- 差距越大，转化越好

## 定价策略技巧

### 1. 心理定价
- $19.99 vs $20
- $97 vs $100
- $497 vs $500

### 2. 套餐定价
- 基础版 $29
- 进阶版 $49 (最佳值)
- 高级版 $99

### 3. 赠品定价
- "价值$500的 bonuses"
- 数量限定的紧迫感

### 4. 分期付款
- 降低购买阻力
- "$49/月 × 3个月"

## 涨价策略

### 1. 自然涨价
- 新功能 = 新价格
- 年度更新版本

### 2. 阶梯涨价
- $49 → $59 → $69
- 老客户优惠保留

### 3. 捆绑涨价
- 单品 $49
- 捆绑包 $79 (省 $19)

## 场景模板

### 在线课程
- 入门课: $19-$49
- 系统课: $97-$197
- 私教: $497-$997

### SaaS产品
- 免费版 → 付费版 ($9-$29)
- 专业版 ($49-$99)
- 企业版 (联系销售)

### 数字产品
- 模板/清单: $19-$49
- 系统/课程: $97-$197
- 高级Bundle: $297-$497

### 咨询服务
- 初次咨询: $100-$300
- 项目制: $1000-$5000
- 长期顾问: $2000+/月
