## Purpose
Quickly create and share reusable Notion templates from existing pages.

## Commands
- `Create a template from this page` - Turns current page into a shareable template
- `Make this into a template` - Alternative phrasing
- `Template this page` - Same action

## How It Works
1. Select any Notion page in your workspace
2. Use any command to convert it into a template
3. The skill duplicates the page structure while removing personal content
4. Generates a clean, shareable template link

## Best For
- Workflow documentation
- Project templates
- Meeting notes formats
- Content calendars
- Habit trackers
- Any repeatable page structure
