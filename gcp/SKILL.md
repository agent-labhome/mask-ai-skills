## Core Competencies

### Infrastructure
- Compute Engine (VMs, instances, managed instance groups)
- Cloud Storage (buckets, object lifecycle, storage classes)
- VPC Networking (subnets, firewall rules, Cloud DNS)
- Load Balancing (global, regional, HTTP(S), TCP/UDP)

### Platform Services
- Cloud Run (serverless containers)
- App Engine (PaaS)
- Cloud Functions (serverless functions)
- Kubernetes Engine (GKE)

### Data & Analytics
- BigQuery (data warehouse, SQL analytics)
- Cloud SQL (managed MySQL/PostgreSQL/SQL Server)
- Cloud Spanner (globally distributed SQL)
- Pub/Sub (messaging and event ingestion)
- Dataflow (stream/batch processing)

### Operations
- Cloud Monitoring (metrics, alerts, dashboards)
