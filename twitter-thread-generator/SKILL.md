# twitter-thread-generator

## Description
Twitter/X Thread生成助手 - 帮助创建病毒式传播的推特长文系列

## Inputs
- 主题/话题
- 目标受众
- 风格偏好 (专业/幽默/励志/教育)
- 推文数量 (3-10条)

## Output Structure
生成完整的推特thread，每条推文包括:
- 内容文案
- 建议的配图/媒体
- 互动引导语 (CTA)

## Features
1. 主题分析 - 提取核心观点和钩子
2. 结构设计 - 开头-中间-结尾的叙事弧线
3. 优化建议 - 提高参与度的技巧
4. Hashtag推荐
5. 发布时机建议

## Workflow
1. 用户输入主题和偏好
2. AI生成thread大纲
3. 逐条生成推文内容
4. 优化语言风格
5. 添加配图建议和CTA

## Example
输入: "AI创业机会"
输出: 5条推文的thread，从AI趋势分析到具体创业建议

## Usage
```
生成一个关于[主题]的thread
- 受众: [描述]
- 风格: [风格]
- 数量: [X条]
```
