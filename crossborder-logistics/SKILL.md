# CrossBorder-Logistics
## 技能描述

**功能**: 跨境物流方案规划助手，帮你选择最优物流渠道

**使用场景**: 
- 跨境电商发货
- 海外仓配置
- 物流成本优化

**触发词**: 跨境物流,国际快递,物流方案

**价格**: 免费

---

## 核心能力

1. **渠道对比** - 各物流商优劣势
2. **成本计算** - 运费和时效分析
3. **清关指导** - 报关和关税建议
4. **海外仓** -仓储和一件代发

---

## 使用方法

提供货物信息、目的地和时效要求，AI推荐最优方案。

## 输出示例

```
📦 物流方案 (美国):

✈️ 空运 (3-5天):
- DHL: $25/kg
- FedEx: $22/kg
- 推荐: FedEx

🚢 海运 (25-35天):
- 美森: $3.5/kg
- 以星: $2.8/kg
- 推荐: 美森

🏭 海外仓:
- 推荐: 4PX仓
- 费用: $2/件/月
- 一件代发: +$1/件
```
