# 中文内容创作助手

## 功能

生成高质量中文内容，适用于各大社交媒体平台。

## 使用场景

### 1. 小红书笔记
- 种草文案
- 产品测评
- 生活分享
- 教程攻略

### 2. 抖音脚本
- 短视频脚本
- 口播文案
- 直播话术

### 3. 知乎回答
- 专业问答
- 经验分享
- 深度分析

### 4. 微信公众号
- 文章写作
- 推文撰写

## 参数

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| platform | string | 是 | 目标平台 (xiaohongshu/douyin/zhihu/wechat) |
| topic | string | 是 | 内容主题 |
| tone | string | 否 | 语气 (专业/活泼/亲和/幽默) 默认活泼 |
| length | string | 否 | 长度 (短/中/长) 默认中 |
| include_hashtags | boolean | 否 | 是否包含标签 默认true |
