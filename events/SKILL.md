## Purpose
Handle event-driven programming, including event creation, subscription, dispatch, and lifecycle management.

## Core Capabilities

### 1. **Event System**
- Create custom events with typed payloads
- Subscribe/unsubscribe to events
- Dispatch events with data
- Event bubbling/capturing support
- Prevent default behavior

### 2. **Event Types**
- DOM events (click, input, submit)
- Custom application events
- System events (resize, scroll, visibility)
- Keyboard/mouse events
- Touch/gesture events

### 3. **Event Management**
- Event delegation
- Event throttling/debouncing
- One-time event listeners

## 使用场景
本技能适用于需要帮助的用户，可应用于多种工作场景。
