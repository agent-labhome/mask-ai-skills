# Skill: 跨境物流成本计算器

## 触发词
- "物流成本计算"
- "shipping cost"
- "跨境运费"
- "物流报价"

## 功能描述
计算跨境电商物流成本，比较不同物流渠道。包含快递、空运、海运、铁路费用计算，关税估算，最优方案推荐。

## 使用场景
1. 产品定价参考
2. 物流渠道选择
3. 库存仓储规划
4. 大货运输优化
5. 成本利润核算
