# Browser Automation Scraper

## 技能描述

构建自动化浏览器脚本，实现网页数据采集、批量操作、表单填写等重复任务自动化。支持反检测和代理轮换。

## 触发词

- "爬虫"
- "数据采集"
- "scraper"
- "自动化脚本"
- "批量操作"
- "网页抓取"

## 功能说明

1. **数据采集** - 从任何网站提取结构化数据
2. **自动化流程** - 批量注册、发布、填写表单
3. **反检测** - 浏览器指纹和代理IP管理
4. **定时任务** - 设置自动执行计划
5. **数据导出** - JSON, CSV, Airtable等格式
6. **监控告警** - 数据变化时自动通知

## 使用场景

- 竞品价格监控
- 潜在客户信息采集
- 社交媒体批量操作
- 电商库存监控
