## 使用示例

```
用户: 我想写一篇SEO文章

助手:
## ✍️ SEO内容写作指南

### 关键词布局

**目标关键词:** "远程工作"

**分布:**
- 标题 (H1): "远程工作完整指南 [2024]"
- H2: "什么是远程工作"
- H3: "远程工作的优势"
- H3: "如何开始远程工作"
- URL: /yuan-gong-zuo/
- 首段: 前50词出现
- 结尾: 总结+关键词

### SEO文章结构

```
[H1] 远程工作完整指南

[介绍] 什么是远程工作，为什么流行

[H2] 远程工作的优势
[H3] 1. 时间灵活
[H3] 2. 成本节省
[H3] 3. 更好的工作生活平衡

[H2] 如何开始远程工作
[H3] 寻找远程工作机会
[H3] 准备远程工作设备
[H3] 建立远程工作习惯

[H2] 远程工作工具推荐

[H2] 常见问题 (FAQ)
[问题] 远程工作真的可行吗?
[答案] ...

[总结]
```

### 关键词密度
- 1-2% 为宜
- 自然分布
- 不要堆砌

### 元描述模板
```
[数字]个[关键词]技巧，帮助[目标]。了解[好处]，立即[行动]。[年份]最新指南。
```

### 图片优化
- 压缩: TinyPNG
- Alt: 描述性关键词
- 文件名: 关键词.png
```
