## Overview
CodeBuddy是腾讯云推出的AI代码助手，帮助开发者提升编程效率。本Skill提供从入门到高级使用的完整指南。

## Core Functions

### 1. 基础入门
- CodeBuddy账号注册与激活
- IDE插件安装（VS Code, JetBrains等）
- 基础配置与设置
- 支持语言和框架

### 2. 核心功能使用
- AI代码补全
- 自然语言生成代码
- 代码解释与重构
- Bug查找与修复
- 单元测试生成

### 3. 高级技巧
- 自定义提示词模板
- 企业版团队协作配置
- API集成到自有产品
- 与腾讯云其他服务联动

### 4. 场景应用
- 前端开发 (React, Vue, 小程序)
- 后端开发 (Node.js, Python, Java)
- 移动端开发
- AI/机器学习项目
