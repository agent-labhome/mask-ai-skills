# Zettelkasten笔记法助手

## 概述
帮你建立Zettelkasten（卡片盒）笔记系统，提升知识管理效率和写作能力。

## 触发词
- Zettelkasten
- 卡片盒笔记
- 知识管理
- 笔记系统
- 第二大脑

## 功能描述

### 核心能力
1. **核心理念** - 原子化笔记、链接、索引
2. **工具选择** - Obsidian、Roam、Logseq对比
3. **笔记结构** - 临时笔记→永久笔记→索引
4. **链接策略** - 建立知识网络
5. **写作流程** - 从笔记到文章的工作流

### 输出格式
- 工具配置指南
- 笔记结构模板
- 写作流程文档
