# Sales Funnel

帮助你构建销售漏斗，从流量到成交。

## 功能
- 自动执行任务
- 提供专业建议
- 生成优化方案

## 核心能力

### 1. 漏斗设计
- 流量入口
- 引导阶段
- 转化节点
- 成交闭环

### 2. 渠道选择
- 社交媒体
- 搜索引擎
- 邮件营销
- 内容营销

### 3. 自动化
- 邮件序列
-  chatbots
- 跟进系统
- CRM整合

### 4. 优化分析
- 数据追踪
- A/B测试
- 转化优化
- 提升ROI

## 使用示例

```
用户: 我想做一个线上课程的漏斗

助手:
## 🎯 销售漏斗设计

### 漏斗结构

**第一层: 认知 (1000人)**
- 渠道: TikTok/YouTube Shorts
- 内容: 免费干货短视频
- 目标: 吸引关注

**第二层: 兴趣 (100人)**
- 渠道: 邮件列表/社群
- 内容: 免费资源/电子书
- 目标: 收集邮箱

**第三层: 考虑 (10人)**
- 渠道: 邮件序列/网页
- 内容: 免费课程/直播
- 目标: 展示价值

**第四层: 成交 (1人)**
- 渠道: Landing Page
- 内容: 限时优惠/保证
- 目标: 完成购买

### 邮件序列示例

**Day 1: 欢迎邮件**
"欢迎！这是你的免费资源..."

**Day 3: 价值展示**
"3个技巧帮你..."

**Day 7: 社会证明**
"学员案例..."

**Day 14: 限时优惠**
"课程优惠最后48小时..."
```
