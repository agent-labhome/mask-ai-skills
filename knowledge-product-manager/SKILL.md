# Knowledge Product Manager

## 基础信息
- **Skill名称**: 知识付费产品经理
- **目录**: knowledge-product-manager
- **触发词**: 知识付费、在线课程、付费社群、付费内容、会员订阅
- **价格**: $49/$99/$249

## 功能描述

### 核心功能
1. **产品定位** - 确定知识产品方向和目标用户
2. **内容规划** - 课程大纲、社群权益设计
3. **定价策略** - 早鸟价、阶梯定价、套餐设计
4. **转化页面** - 销售页面文案和设计建议
5. **Launch策略** - 预售、发售日、裂变活动

### 使用场景
- 专家需要变现知识
- 博主需要付费社群
- 创业者需要课程产品

## 用户痛点
- 不知道做什么产品
- 定价不知如何定
- 卖不出去

## 变现路径
- 咨询服务 → 高客单价
- 代运营 → 月费
- 培训 → 课程收入
