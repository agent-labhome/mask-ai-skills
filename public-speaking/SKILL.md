## 使用示例

```
用户: 我演讲会很紧张

助手:
## 🎤 演讲指南

### 开场hook

**问题开场:**
"你有没有想过..."

**数据开场:"
"90%的人不知道..."

**故事开场:**
"3年前，我..."

**震撼开场:**
"这是我见过最..."

### 演讲结构

**PREP:**
- Point: 观点
- Reason: 理由
- Example: 例子
- Point: 总结

**故事结构:**
- 背景
- 冲突
- 高潮
- 结局

### 克服紧张

**准备:**
- 练习20遍以上
- 熟悉内容
- 准备应急预案

**现场:**
- 深呼吸3次
- 看熟悉的面孔
- 放慢语速

**心态:**
- 接受紧张
- 专注于内容
- 享受分享

### 肢体语言

- 站姿: 稳
- 手势: 自然
- 眼神: 扫视
- 表情: 真诚
```
