# Viral Growth

## Definition
Viral growth is a growth pattern where existing users of a product or service actively and organically recruit new users, creating a self-perpetuating cycle of adoption. This is typically measured by the **viral coefficient (K-factor)**, which quantifies how many new users each existing user brings in.
