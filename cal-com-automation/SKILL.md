# Cal Com Automation

**触发词**: Cal.com自动化, 日程管理, 预约系统, 日历集成

**功能描述**: 
Cal.com是开源的日程预约平台。本技能帮助用户：
- 设置预约页面和工作流
- 集成Zoom、Google Calendar
- 设计确认和提醒邮件
- 配置团队调度规则
- 分析预约数据

**使用场景**:
- 咨询: 顾问、教练预约
- 销售: 演示、咨询预约
- 医疗: 诊所、预约系统
- 教育: 课程、辅导预约
