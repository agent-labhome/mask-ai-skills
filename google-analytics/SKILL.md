## Core Concepts

### 1. **Measurement Model**
- **Users**: Unique visitors to your site/app
- **Sessions**: Individual visits (can include multiple pageviews)
- **Pageviews**: Total number of pages viewed
- **Events**: User interactions (clicks, downloads, video plays)
- **Conversions**: Goal completions (purchases, sign-ups, etc.)

### 2. **Key Reports**
- **Audience**: Who visits your site (demographics, devices, locations)
- **Acquisition**: How users find you (organic search, social, direct, referral)
- **Behavior**: What users do on your site (pageviews, site speed, content)
- **Conversions**: Business outcomes (goals, e-commerce, funnels)

## Practical Implementation Guide

### Setup Phase (Week 1)
1. **Account & Property Creation**
   - Create Google Analytics account
   - Set up property for your website/app
   - Install tracking code (gtag.js or Google Tag Manager)

2. **Basic Configuration**
   - Set up data filters (exclude internal traffic)
   - Configure time zone and currency
   - Link to Google Search Console
   - Set up basic goals (contact form submissions, purchases)
