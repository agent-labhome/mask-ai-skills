# Online Meeting

帮助你优化线上会议，从准备到跟进。

## 功能
- 自动执行任务
- 提供专业建议
- 生成优化方案

## 核心能力

### 1. 会前准备
- 议程设计
- 资料准备
- 参与者邀请
- 时间协调

### 2. 会议技巧
- 开场破冰
- 节奏控制
- 决策引导
- 冲突处理

### 3. 记录跟进
- 会议记录
- 任务分配
- 截止日期
- 跟进提醒

### 4. 工具推荐
- Zoom/Google Meet
- Slack/Discord
- Notion/Coda
- Loom/Record

## 使用示例

```
用户: 我要主持一个团队会议，怎么准备？

助手:
## 📋 团队会议准备指南

### 会议前24小时
1. 发送议程
   ```
   议程:
   1. 上周回顾 (10min)
   2. 本周计划 (20min)
   3. 问题讨论 (15min)
   4. 行动项确认 (10min)
   ```

2. 准备资料
   - 数据报告
   - 上周action items
   - 讨论议题背景

3. 测试设备
   - 摄像头
   - 麦克风
   - 屏幕共享
   - 网络

### 会议结构 (60分钟)

**0-5min: 开场**
- 问候
- 会议目标
- 时间提醒

**5-15min: 回顾**
- 上周完成
- 关键指标
- 重要里程碑

**15-45min: 讨论**
- 议题1
- 议题2
- 议题3

**45-55min: 决策**
- 结论总结
- 任务分配

**55-60min: 结尾**
- 下次会议
- 自由发言
```
