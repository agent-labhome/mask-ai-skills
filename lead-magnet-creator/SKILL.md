# Lead Magnet Creator

创建高质量的引流赠品(Lead Magnet)来建立你的邮件列表。

## Lead Magnet类型

1. **电子书(E-books)** - 深度指南或教程
2. **清单(Checklists)** - 可执行的待办事项列表
3. **模板(Templates)** - 可直接使用的工具或框架
4. **清单(Cheat Sheets)** - 快速参考指南
5. **视频教程** - 演示类内容
6. **案例研究** - 成功故事和分析
7. **工作表(Worksheets)** - 互动练习材料

## 创建流程

### 1. 确定目标受众痛点

- 目标用户最大的3个问题是什么？
- 他们现在如何解决这些问题？
- 什么信息能真正帮助他们？

### 2. 选择Lead Magnet格式

根据内容类型选择：
- 复杂概念 → 电子书
- 需要行动 → 清单/工作表
- 需要工具 → 模板
- 视觉学习 → 视频教程

### 3. 撰写标题

好标题的特征：
- 具体且有吸引力
- 明确承诺价值
- 包含数字或结果

示例：
- "7天获取1000个邮件订阅"
- "2024年完整SEO清单"
- "ChatGPT营销实战模板"

### 4. 设计内容结构

保持简洁，聚焦一个核心问题：
- 问题定义
- 解决方案步骤
- 可执行的行动项

### 5. 转化优化

- 落地页标题：明确告诉用户将获得什么
- 表单：只收集必要信息（邮箱必填，其他可选）
-  CTA：使用行动导向语言

## 输出格式建议

### 电子书结构
```
标题
简介
第1章：问题定义
第2章：解决方案
第3章：实操步骤
结论
行动号召
```

### 清单结构
```
标题
使用说明
项目1
项目2
项目3
...
```

### 模板结构
```
模板介绍
使用场景
模板内容
示例
注意事项
```
