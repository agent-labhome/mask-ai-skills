# SKILL: Online Course Creation

## Core Competencies
- **Curriculum Design**: Structuring learning paths from beginner to advanced
- **Content Development**: Creating engaging lessons, assignments, and assessments
- **Multimedia Production**: Recording videos, designing slides, creating graphics
- **Platform Proficiency**: Using LMS platforms (Thinkific, Teachable, Kajabi, etc.)
- **Learning Psychology**: Applying instructional design principles for retention

## Key Deliverables
- Course outline with modules and learning objectives
- Video lessons (screencasts, talking head, animations)
- Downloadable resources (worksheets, templates, checklists)
- Quizzes and assessments
- Community/engagement strategies

## Best Practices
- Start with audience research and problem identification
- Chunk content into 5-15 minute segments
- Mix theory with practical application
- Include interactive elements
- Plan for updates and revisions
