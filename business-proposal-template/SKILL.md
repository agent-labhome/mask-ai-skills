# Business-Proposal-Template
## 技能描述

**功能**: 商业提案/报价单生成器，专业打动客户

**使用场景**: 
- 客户提案
- 项目报价
- 合作邀请

**触发词**: 商业提案,报价单,proposal

**价格**: 免费

---

## 核心能力

1. **结构设计** - 专业提案框架
2. **内容撰写** - 价值主张
3. **定价策略** - 报价技巧
4. **跟进技巧** - 成交策略

---

## 输入格式

```
客户类型:
项目类型:
预算范围:
```
