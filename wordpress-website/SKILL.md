## Purpose
Quickly create and manage WordPress websites with automated setup, content generation, and maintenance.

## Commands
- `create wordpress site` - Set up a new WordPress installation
- `add content` - Generate and publish posts/pages
- `install plugin [name]` - Install and configure plugins
- `optimize site` - Improve performance and security
- `update wordpress` - Update core, themes, and plugins
- `backup site` - Create complete website backup
- `fix issue` - Troubleshoot common WordPress problems

## Capabilities
- One-click WordPress installation
- Automated content creation
- Plugin/theme management
- SEO optimization
- Security hardening
- Performance tuning
- Database maintenance
- Backup/restore operations
