# 跨境电商运营助手

## 功能

专为跨境电商设计的AI助手，帮助你完成选品、文案、客服等工作。

## 使用场景

### 1. 选品分析

**输入:** 产品品类 + 目标市场

**输出:**
- 市场分析
- 竞争度评估
- 利润预估
- 推广建议

### 2. 商品文案

**输入:** 产品信息 + 目标平台

**输出:**
- 标题 (含关键词)
- 卖点描述
- 详情页文案
- 买家秀文案

### 3. 客服回复

**输入:** 买家问题

**输出:**
- 友好回复
- 解决问题方案
- 关联产品推荐

### 4. 市场调研

**输入:** 想做的品类

**输出:**
- 市场规模
- 趋势分析
- 热门关键词
- 竞品分析

## 参数

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| task | string | 是 | 任务类型 (选品/文案/客服/调研) |
| product | string | 是 | 产品信息 |
| platform | string | 是 | 目标平台 |
| market | string | 否 | 目标市场 |
| language | string | 否 | 输出语言 |
