# 跨境电商运营助手

## 概述

本技能帮助运营跨境电商业务，覆盖选品、Listing优化、物流、客服等全流程。

## 选品分析

### 选品维度

1. **市场需求**: 搜索量、趋势、季节性
2. **竞争程度**: 卖家数量、Review数量、垄断程度
3. **利润空间**: 成本、运费、平台费用、目标利润率>30%
4. **物流可行性**: 尺寸、重量、是否易碎、是否敏感货
5. **合规要求**: 认证、专利、出口限制

### 选品工具

- Jungle Scout / Helium 10 / AMZ123
- Google Trends 查趋势
- 亚马逊BSR榜单对标

### 选品公式

```
利润率 = (售价 - 成本 - 物流 - 平台费) / 售价
目标: >30% 纯利
```

## 平台运营

### 亚马逊 (Amazon)

- **FBA vs FBM**: 推荐FBA获取Prime流量
- **Listing优化**: 标题、五点描述、Search Term、QA
- **关键词调研**: 竞品词库、ABA数据、长尾词
- **PPC广告**: 自动/手动广告、ACOS目标<25%
- **Review获取**: Vine计划、真实购买评论

### Shopee / Lazada (东南亚)

- **Shopee Live**: 直播带货权重高
- **母婴/3C/时尚**: 东南亚热销品类
- **物流**: Shopee Logistics Service (SLS)

### eBay

- **二手/翻新**: eBay特色品类
- **国际航运**: eBay全球运输计划

## 物流方案

### 头程物流

- **空派**: 7-15天，适合轻小件、季节品
- **海派**: 25-40天，适合大件、低价值商品
- **铁派**: 中欧班列，20-25天

### 仓储方案

- **亚马逊FBA**: 头程+入仓一体化
- **第三方海外仓**: 退货处理、一件代发

### 成本计算

```
单件成本 = 商品成本 + 头程运费 + 平台佣金(15%) + FBA费用 + 推广费
```

## 文案撰写

### 五点描述模板

1. 痛点场景
2. 产品特点
3. 材质/规格
4. 使用方法
5. 售后保障

### 英文文案要点

- 关键词自然嵌入
- 情感化表达
- 数据化佐证 (如"10000+ sold")
- 行动号召 (CTA)

## 支付与结算

### 回款周期

- 亚马逊: 14-21天
- Shopee: T+7
- PayPal: 3-7天

### 汇率管理

- 建议锁汇或使用多货币账户
- 关注汇率波动，适时提现

## 客服处理

### 常见问题

- **物流延迟**: 主动告知、适当补偿
- **产品损坏**: 补发或退款
- **差评**: 诚恳沟通、解决后请删评
- **A-to-Z索赔**: 积极响应、提供证据

### 邮件模板

```

Thank you for reaching out. We're sorry to hear about the issue with your order.

[针对具体问题的解决方案]

We value your business and hope to serve you better in the future.

Best regards,
[Store Name]
```

## 竞争对手分析

### 分析维度

1. 定价策略
2. 流量来源 (广告/自然/关联)
3. Review数量和质量
4. Listing优化水平
5. 促销节奏

### 监控工具

- Keepa (亚马逊历史数据)
- Jungle Scout
- 谷歌搜索监控

## 爆款打造

### 新品推广节奏

1. **Week 1-2**: 优化Listing、积累首批Review
2. **Week 3-4**: 启动PPC、适量测评
3. **Month 2**: 优化广告、站外引流
4. **Month 3+**: 稳定排名、拓展变体

### 站外引流

- Facebook/Instagram广告
- TikTok短视频
- Deals网站 (Slickdeals, Dealsplus)
- Influencer合作
