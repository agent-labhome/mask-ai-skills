# SKILL.md: Freelance Proposal Brief

## Purpose
A concise document that outlines project requirements, scope, and expectations to solicit accurate proposals from freelancers.

## Core Components

### 1. Project Overview
- **Title**: Clear project name
- **Objective**: What you want to achieve (1-2 sentences)
- **Problem Statement**: The challenge you're solving

### 2. Scope & Deliverables
- **Specific tasks** or features needed
- **Tangible outputs** (e.g., website, report, design files)
- **Technical requirements** (platforms, integrations, formats)

### 3. Timeline & Budget
- **Deadline**: Desired completion date
- **Milestones**: Key checkpoints (if applicable)
- **Budget Range**: Hourly rate or project-based expectations
