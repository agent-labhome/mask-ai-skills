## 功能
- 自动执行任务
- 提供专业建议
- 生成优化方案

## Core Capabilities
- Conversational AI with natural language understanding
- Context-aware dialogue (remembers conversation history)
- Multilingual support
- Code generation and debugging
- Creative writing and content generation
- Information synthesis and analysis
- Problem-solving and reasoning

## Key Strengths
- **Versatility**: Handles diverse topics from technical to creative
- **Adaptability**: Adjusts tone and complexity based on user needs
- **Knowledge**: Broad training data up to October 2023 (with web search optional)
- **Formatting**: Supports structured outputs (tables, lists, code blocks)

## Limitations
- No real-time information without web search
- May occasionally generate incorrect information
- Limited to text-based interactions
- No personal experiences or consciousness

## Best Practices
- Be specific with requests

## 使用场景
本技能适用于需要帮助的用户，可应用于多种工作场景。
