# Teachable在线课程创建

## 功能

帮助在Teachable平台创建和销售在线课程，提供从课程设计到营销的完整指导。

## 使用场景

### 1. 课程设计
- 课程大纲规划
- 内容结构设计
- 价格策略制定

### 2. 制作上传
- 视频录制指导
- 文档资料整理
- 测验题目设置

### 3. 营销推广
- Landing Page优化
- 折扣码设置
- 联盟分销设置

## 参数

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| subject | string | 是 | 课程主题 |
| course_level | string | 是 | 课程级别 (beginner/intermediate/advanced) |
| student_goal | string | 是 | 学生目标 |
| price | number | 是 | 课程定价 |
