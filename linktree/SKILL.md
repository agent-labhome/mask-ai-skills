## Purpose
Quickly create and manage Linktree-style link pages.

## Commands
- `create linktree` - Start a new Linktree page
- `add link [title] [url]` - Add a link to your page
- `preview` - See how your Linktree looks
- `publish` - Get a shareable URL
- `edit link [number]` - Modify an existing link
- `remove link [number]` - Delete a link

## Features
- Customizable page title and bio
- Social media icon support
- Theme selection (light/dark/custom)
- Analytics tracking
- Mobile-responsive design

## Tips
- Keep link titles short and clear
- Use emojis in titles for visual appeal
- Group similar links together
- Update regularly with fresh content
