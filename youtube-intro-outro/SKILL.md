### 开场 (5-10秒)

**结构:**
```
0-2秒: 视觉hook
2-5秒: 自我介绍/频道名
5-10秒: 本期内容预告
```

**模板:**
```
[动画Logo]
[频道名]
[欢迎大家，我是xxx]
[今天来聊聊xxx]
```

### 结尾 (10-20秒)

**结构:**
```
0-5秒: 本期总结
5-15秒: CTA (订阅+点赞)
15-20秒: 相关推荐+结束画面
```

**模板:**
```
[感谢观看]
[如果喜欢，请订阅]
[下期预告]
[End Card]
```
