# Notion自动化工作流

## 功能

帮助搭建Notion自动化工作流，连接其他工具实现数据同步和流程自动化。

## 使用场景

### 1. 项目管理自动化
- 任务状态自动更新
- 截止日期提醒
- 进度追踪自动化

### 2. 数据同步
- Notion与Google Calendar同步
- Slack通知集成
- 邮件自动归档

### 3. 内容工作流
- 社交媒体排程
- 博客发布自动化
- 客户CRM管理

## 参数

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| workflow_type | string | 是 | 工作流类型 |
| trigger_app | string | 是 | 触发应用 |
| action_app | string | 是 | 动作应用 |
| complexity | string | 是 | 复杂度 (simple/medium/complex) |
