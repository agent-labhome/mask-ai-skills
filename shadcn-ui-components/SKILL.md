# Shadcn Ui Components

## 触发词
shadcn ui, react组件, tailwind组件, UI组件库

## 功能描述
shadcn/ui是基于Radix UI和Tailwind CSS的现代React组件库，提供可定制、可复制的组件。

## 使用场景
- React项目UI开发
- 快速原型构建
- 企业级应用界面
- 设计系统搭建
