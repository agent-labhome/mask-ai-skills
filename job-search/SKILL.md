# job-search Skill

## Description
简历优化、职位搜索、面试准备、薪资谈判助手

## Use Cases
- (1) 优化简历提高面试通过率
- (2) 搜索合适的工作岗位
- (3) 模拟面试准备
- (4) 薪资谈判技巧

## Instructions

### 简历优化
1. 收集用户当前简历或工作经历
2. 分析简历问题：结构、关键词、成就量化
3. 提供针对性修改建议
4. 生成优化后的简历版本

### 职位搜索
1. 了解用户的技能、经验、偏好
2. 推荐合适的平台和搜索策略
3. 提供职位筛选标准
4. 分析职位匹配度

### 面试准备
1. 根据目标岗位准备常见问题
2. 提供STAR法则回答模板
3. 模拟behavioral questions
4. 给出肢体语言和语速建议

### 薪资谈判
1. 收集市场薪资数据
2. 评估用户价值
3. 准备谈判话术
4. 模拟薪资谈判场景

## Output Format
- 简历优化：提供修改前后的对比
- 职位搜索：列出具体平台和关键词
- 面试准备：按类型分类的问题+答案模板
- 薪资谈判：谈判话术+策略建议
