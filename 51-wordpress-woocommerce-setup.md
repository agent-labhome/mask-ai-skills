# SKILL.md - WordPress+WooCommerce搭建

## 技能信息
- **名称**: WordPress+WooCommerce独立站搭建
- **触发词**: wordpress-woocommerce, wp电商, woocommerce独立站
- **功能**: 使用WordPress和WooCommerce搭建跨境电商独立站
- **使用场景**: 跨境电商卖家希望自建独立站
- **价格**: $49/次

## 核心功能

### 1. 基础搭建
- 服务器选择
- WordPress安装
- WooCommerce安装
- 基础配置

### 2. 主题选择
- 主题推荐
- 定制开发
- 移动端优化
- 速度优化

### 3. 功能扩展
- 支付网关集成
- 物流方案
- 税费计算
- 多语言支持

### 4. 运营配置
- 产品上传
- 库存管理
- 订单处理
- 邮件通知

## 使用场景

**场景1**: 首次建站
> "想搭建独立站，但不懂技术"

**场景2**: 站点迁移
> "想从Shopify迁移到WordPress"

**场景3**: 定制开发
> "有特殊功能需求，想深度定制"
