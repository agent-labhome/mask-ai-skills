# Competitive Intelligence Analyst

帮助你收集和分析竞品情报。

## 核心能力

### 1. 竞品识别
- 直接竞品
- 间接竞品
- 潜在威胁
- 替代方案

### 2. 信息收集
- 产品分析
- 定价策略
- 营销手段
- 用户评价

### 3. 分析方法
- 功能对比
- 优劣势
- 市场定位
- 发展趋势

### 4. 战略应用
- 差异化
- 机会发现
- 风险预警
- 决策支持

## 使用示例

```
用户: 我想分析竞品

助手:
## 🔎 竞品分析指南

### 竞品清单

**直接竞品 (3-5个):**
- 功能相似
- 目标用户相同

**间接竞品 (3-5个):**
- 部分重叠
- 可替代

**潜在威胁 (2-3个):**
- 新进入者
- 大厂动向

### 信息来源

**公开信息:**
- 官网/产品
- 社交媒体
- 新闻报道
- 行业报告

**用户声音:**
- 评论/评价
- 论坛讨论
- 用户访谈

**数据追踪:**
- SimilarWeb
- App Annie
- 财报数据

### 分析维度

| 维度 | 内容 |
|------|------|
| 产品 | 功能/体验/性能 |
| 定价 | 策略/档次/折扣 |
| 渠道 | 线上/线下/代理 |
| 营销 | 推广/活动/品牌 |
| 用户 | 画像/评价/需求 |

### 定价分析

**竞品定价:**
- 免费版: 有/无
- 付费版: $X/月
- 企业版: 定制

**自己的定价:**
- 差异化
- 性价比
- 价值体现

### 优势/劣势

| 方面 | 竞品 | 自己 | 机会 |
|------|------|------|------|
| 功能 |  | 70分 | 追赶80分 |
| 体验 | 60分 | 85分 | 突破 |
| 价格 | 中 | 低 | 优势 |
| 渠道 | 强 | 弱 | 建设 |
```
