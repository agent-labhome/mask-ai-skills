## Overview
Streaming is the real-time delivery of digital content (audio, video, games, data) from a source to an end-user device over a network, without requiring the user to download the entire file first.

## Core Concepts
- **Live Streaming:** Real-time broadcast (e.g., Twitch, live sports).
- **Video on Demand (VOD):** Pre-recorded content (e.g., Netflix, YouTube).
- **Latency:** Delay between capture and playback.
- **Bitrate:** Data transferred per second (affects quality).
- **Buffering:** Preloading content to prevent interruptions.
- **Codec:** Compression standard (e.g., H.264, AV1).

## Key Platforms & Services
- **Live:** Twitch, YouTube Live, Facebook Live, TikTok Live.
- **VOD:** Netflix, Hulu, Disney+, Amazon Prime Video.
- **Music:** Spotify, Apple Music, Tidal.
