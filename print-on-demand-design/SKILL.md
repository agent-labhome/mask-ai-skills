# Print-on-Demand-Design
## 技能描述

**功能**: Print-on-Demand设计灵感生成器，帮你创作畅销设计

**使用场景**: 
- T恤/周边产品设计
- Etsy/Redbubble销售
- 被动收入

**触发词**: POD设计,t恤设计,周边产品

**价格**: 免费

---

## 核心能力

1. **趋势分析** - 热门 niches 和关键词
2. **设计方向** - 配色和风格建议
3. **文案优化** - 畅销文案参考
4. **平台策略** - 各平台优化技巧

---

## 输入格式

```
目标平台:
产品类型:
风格偏好:
```

