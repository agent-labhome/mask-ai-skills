# Customer Journey Map

## 触发词
- "客户旅程"
- "用户旅程"
- "customer journey"
- "用户路径"

## 功能描述
绘制完整的客户旅程地图，优化转化漏斗。

## 使用场景
1. 转化漏斗优化
2. 用户体验改进
3. 营销触点梳理
4. 客户留存策略

## 价格
$29

## 核心能力
- 阶段划分：认知 → 考虑 → 决策 → 留存 → 推荐
- 触点分析：线上/线下/社交/客服全渠道
- 痛点识别：使用Jobs-to-Be-Done方法
- 优化建议：基于AARRR模型的具体方案

## 实施步骤

### 1. 调研阶段
- 用户访谈（至少5人）
- 数据分析（行为日志、转化率）
- 竞品分析

### 2. 阶段划分
```
认知( Awareness ) → 考虑( Consideration ) → 决策( Decision ) → 留存( Retention ) → 推荐( Referral )
```

### 3. 触点映射
| 阶段 | 触点 | 渠道 | 用户目标 | 痛点 |
|------|------|------|----------|------|
| 认知 | 广告/搜索 | Google/社媒 | 解决问题 | 信息过载 |
| 考虑 | 官网/评测 | SEO/YouTube | 比较方案 | 选择困难 |
| 决策 | 价格页/评论 | Landing/亚马逊 | 信任证明 | 风险担忧 |
| 留存 | 邮件/社群 | 自动化/Discord | 持续价值 | 遗忘 |
| 推荐 | 评价/邀请 | 口碑/联盟 | 获得奖励 | 动机不足 |

### 4. 痛点分析模板
- **功能痛点**：产品功能不足
- **流程痛点**：操作繁琐
- **情感痛点**：焦虑/不信任
- **社会痛点**：Peer pressure

### 5. 优化输出
- 优先级矩阵（影响×难度）
- 快速胜利清单
- 长期改进路线图
- KPI指标建议
