# SKILL.md - YouTube Manager

## 描述
YouTube频道运营助手 - 帮你管理YouTube频道、制作视频、增长订阅者、优化SEO。

## 使用场景
1. YouTube频道定位和内容规划
2. 视频脚本和创意策划
3. 订阅者增长策略
4. YouTube SEO优化
5. 视频数据分析
6. 变现策略

## 核心功能

### 1. 频道定位
- 目标受众分析
-  niche选择和差异化
- 频道品牌定位
- 内容策略制定

### 2. 内容策划
- 视频主题创意
- 标题和描述优化
- 标签策略
- 缩略图设计建议

### 3. 增长策略
- 观看时长优化
- 订阅者转化
- 互动策略
- 合作和跨推广

### 4. SEO优化
- 关键词研究
- 视频SEO
- 播放列表优化
- 频道SEO

### 5. 数据分析
- 视频表现分析
- 观众留存分析
- 流量来源分析
- 竞品分析

### 6. 变现策略
- YouTube Partner Program
- 品牌合作
- 商品销售
- 会员和超级留言

## 输出格式

根据用户需求，提供:
- 频道运营建议
- 视频创意和脚本
- SEO优化方案
- 数据分析报告
- 增长策略

## 使用示例

"帮我规划一个科技类YouTube频道的内容"
"优化我这个视频的标题和描述"
"如何提高视频的观看时长"
"分析我频道的数据表现"
