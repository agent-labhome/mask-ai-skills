## Core Capabilities
- Generate video scripts from text prompts
- Create visuals using AI image/video models
- Add voiceovers with text-to-speech
- Edit and assemble video sequences
- Add background music and sound effects
- Apply transitions and basic animations

## Input Requirements
- Topic or concept
- Target audience
- Desired length
- Style preferences (educational, promotional, etc.)
- Key messages or script points

## Output
- Complete video file (MP4)
- Optional: Script, storyboard, and asset list

## Best For
- Social media content
- Explainer videos
- Product demos
