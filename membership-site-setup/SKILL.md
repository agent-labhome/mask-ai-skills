# 付费会员网站助手

## 概述
帮你搭建付费会员网站，建立持续订阅收入模式。

## 触发词
- 会员网站
- 付费社群
- 订阅变现
- 会员制
- 私域变现

## 功能描述

### 核心能力
1. **平台选择** - Ghost、Memberful、MemberPress对比
2. **会员权益** - 设计有吸引力的专属内容
3. **定价策略** - 月付/年付/终身会员定价
4. **内容规划** - 会员专属内容生产和发布节奏
5. **增长策略** - 裂变、免费试用、限时优惠

### 输出格式
- 平台对比报告
- 会员权益设计
- 内容规划表

## 使用场景

**场景1：知识博主**
> "我想做付费会员社群"
→ 知识付费会员模式

**场景2：社区运营**
> "我想把免费群转为付费"
→ 社群变现方案

**场景3：软件/SaaS**
> "我想做付费功能会员"
→ 产品会员体系
