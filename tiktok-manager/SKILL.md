# TikTok Manager

TikTok运营助手，帮助创建高效的TikTok账号增长策略和内容方案。

## 账号定位

### 定位框架

- **身份定位**: 谁是你(账号人格)
- **内容定位**: 做什么内容(内容 pillar)
- **价值定位**: 给用户带来什么(实用/娱乐/情感)
- **差异化**: 你的独特卖点是什么

### 热门赛道建议

- 知识类(英语/编程/商业)
- 生活vlog
- 美食/探店
- 健身/运动
- 搞笑/娱乐
- 时尚/美妆
- 宠物
- 手工/DIY

## 内容策略

### 内容 pillar

建立3-5个内容支柱:

1. **核心内容**(60%): 你的主要价值
2. **热点内容**(20%): 跟随热门话题
3. **互动内容**(20%): 问答/投票/挑战

### 发布时间

- 最佳时段: 12:00-14:00, 18:00-20:00, 21:00-23:00
- 美国时间晚间: 19:00-23:00
- 保持固定发布时间，建立观众预期

## 短视频脚本

### 黄金3秒法则

开头3秒必须抓住注意力:
- 提问/痛点
- 反常识/惊人事实
- 视觉冲击
- 悬念设置

### 脚本结构

```
[开头3秒] 抓住注意力
[核心内容] 价值传递(15-45秒)
[结尾] CTA互动/关注引导(3-5秒)
```

### 热门格式

- **How-to教程**: "3步教你xxx"
- **清单列表**: "5个xxx技巧"
- **对比/测评**: "xxx vs xxx"
- **故事叙述**: "我如何做到xxx"
- **问答形式**: "有人问我xxx"
- **挑战记录**: "xxx挑战第x天"

## 涨粉策略

### 核心增长公式

优质内容 + 频繁发布 + 互动运营 = 增长

### 互动技巧

- 回复所有评论(前30分钟关键)
- 评论区引导二次互动
- 发起问答/投票
- 与同赛道创作者互动
- DOU+投放策略

### 蹭热点技巧

- 关注热门音乐/挑战
- 参与平台活动
- 跟进时效性话题
- 节日/节气内容

## 算法优化

### 推荐机制

TikTok算法核心指标(权重排序):
1. **完播率**: 完整观看比例
2. **重播率**: 重复观看次数
3. **互动率**: 点赞/评论/分享/收藏
4. **关注转化**: 看后关注比例
5. **账号权重**: 账号历史表现

### 优化技巧

- 视频长度控制在15-60秒
- 节奏紧凑，避免拖沓
- 字幕+音乐增强观看体验
- 结尾引导互动和关注
- 保持内容一致性

## 数据分析

### 关键指标

- 播放量/浏览量
- 完播率(目标>50%)
- 点赞数/评论数/分享数
- 粉丝增长
- 视频表现趋势

### 分析工具

- TikTok分析中心
- 第三方工具(NotJustAnalytics, Penticle等)
