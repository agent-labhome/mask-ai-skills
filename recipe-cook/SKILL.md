# 食谱烹饪助手

## 功能

智能食谱和烹饪指导工具。

## 使用场景

### 1. 菜谱推荐

**输入:** 食材 + 口味

**输出:**
- 菜谱选择
- 步骤详解
- 技巧提示

### 2. 食材采购

**输入:** 菜谱

**输出:**
- 采购清单
- 保存方法
- 替代建议

### 3. 烹饪指导

**输入:** 菜品

**输出:**
- 步骤详解
- 火候控制
- 常见问题

### 4. 营养分析

**输入:** 菜品

**输出:**
- 营养成分
- 卡路里
- 健康建议

## 参数

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| task | string | 是 | 任务类型 |
| ingredients | string | 是 | 食材 |
| cuisine | string | 否 | 菜系 |

## 示例

### 菜谱推荐

**输入:**
```json
{
  "task": "菜谱",
  "ingredients": "鸡胸肉、西兰花",
  "cuisine": "中式"
}
```

**输出:**
```
🍳 推荐菜谱: 蒜香鸡胸肉+清炒西兰花

📝 蒜香鸡胸肉:
1. 鸡胸肉切块，用盐、胡椒、料酒腌制10分钟
2. 热锅倒油，放蒜末炒香
3. 放入鸡胸肉翻炒至变色
4. 加酱油、蚝油调味
5. 出锅撒葱花

⏱️ 时间: 15分钟
🔪 难度: 简单

📝 清炒西兰花:
1. 西兰花切小朵，焯水2分钟
2. 热锅倒油，放蒜末
3. 放入西兰花翻炒
4. 加盐调味即可

💡 技巧: 鸡胸肉不要炒太久，否则会柴
```
