# Drizzle Orm

## 触发词
drizzle orm, typescript数据库, 数据库ORM, sql工具

## 功能描述
Drizzle是轻量级的TypeScript ORM，提供类型安全的SQL查询和极致的性能。

## 使用场景
- TypeScript全栈项目
- Serverless数据库操作
- PostgreSQL/MySQL项目
- 高性能应用
