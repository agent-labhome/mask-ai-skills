# B2B Lead Generation

帮助你获取B2B客户线索，从定位到成交。

## 核心能力

### 1. 客户画像
- ICP定义
- 行业筛选
- 规模判断
- 决策人识别

### 2. 获取渠道
- LinkedIn开发
- 冷邮件
- 行业活动
- 内容营销

### 3. 验证筛选
- 公司调研
- 联系人验证
- 需求匹配
- 预算判断

### 4. 跟进策略
- 跟进节奏
- 价值追加
- 成交信号
- 拒绝处理

## 使用示例

```
用户: 我做B2B软件销售，怎么找客户？

助手:
## 🎯 B2B客户开发指南

### ICP (理想客户画像)
- 行业: SaaS/电商/金融
- 规模: 50-500人
- 年收入: $1M-50M
- 痛点: 增长瓶颈/效率低

### 开发渠道

**1. LinkedIn**
- 关键词搜索: "CTO at SaaS"
- 1st+2nd度人脉
- 每周添加50人
- 个性化邀请

**2. 冷邮件**
- 目标: 100-500人公司
- 找到具体联系人
- 3封序列:
  - Day 1: 价值型
  - Day 3: 案例型
  - Day 7: 紧迫感

**3. 行业活动**
- SaaS大会
- 行业展会
- Webinar

### 验证清单
- [ ] 公司官网存在
- [ ] 有采购需求
- [ ] 有预算
- [ ] 有决策人

### 跟进模板
```
Hi [名字],

看到贵司[具体观察]，我们帮[类似公司]解决了[问题]。

[客户案例]: [结果]

想了解更多，可以预约15分钟免费咨询。

[你的名字]
```
```
