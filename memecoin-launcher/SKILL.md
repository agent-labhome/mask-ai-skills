# Memecoin Launcher

一键创建并发射你的Memecoin。

## 功能
- 代币经济学设计（名称、符号、总供应量、税率）
- 生成部署代码（Solana/ETH）
- 创建代币网站模板
- 社区启动策略

## 输入
- 代币名称和符号
- 区块链网络（Solana/Ethereum/Base）
- 特殊功能（burn、锁仓、LP）

## 输出
- 完整的代币合约代码
- 部署指南
- 营销时间表

## 示例代币
```
Name: PEPEAI
Symbol: PEPEAI
Supply: 1,000,000,000
Tax: 0/0
Network: Solana
```
