# Gumroad产品上线助手

_帮助在Gumroad创建和上线AI技能产品_

## 触发条件

当需要创建Gumroad产品页面时使用此技能。

## 核心功能

### 1. 产品页面结构
- 简洁有力的标题
- 清晰的产品描述
- 定价策略 ($9-$49区间)
- 预览图/封面设计指导

### 2. 上线检查清单
- [ ] 产品标题 (简洁有力)
- [ ] 产品描述 (包含功能、适用人群、优势)
- [ ] 封面图片 (1280x960推荐)
- [ ] 定价设置
- [ ] 分类标签
- [ ] 预览内容设置

### 3. 定价策略建议

**低价走量 ($9-19)**
- 单个技能/工具
- 模板合集
- 快速入门指南

**中价 ($29-49)**
- 技能合集 (20-50个)
- 完整工作流
- 会员/订阅

**高价 ($99+)**
- 定制方案
- 企业授权
- 咨询服务

### 4. 描述模板

```
[产品名称] - [一句话卖点]

🎯 适用人群：
- [人群1]
- [人群2]

✨ 包含内容：
- [内容1]
- [内容2]
- [内容3]

💡 为什么选择这个：
- [优势1]
- [优势2]

🔧 使用要求：
- [要求1]
- [要求2]

立即访问：[Gumroad链接]
```

### 5. 推广素材

创建后生成：
- Twitter/X 推广文案
- LinkedIn帖子
- 邮件推送文案

## 执行步骤

1. 确定产品定位和目标人群
2. 撰写产品描述
3. 准备封面图片
4. 设置价格
5. 在Gumroad创建页面
6. 测试购买流程
7. 启动推广

## 注意事项

- 首次销售建议从低价开始
- 收集用户反馈迭代
- 保持产品页面简洁
- 定期更新内容
