## Core Competencies
- Writing clean, modular Python scripts for automation and data processing
- Using standard library modules (os, sys, json, csv, datetime, pathlib)
- Implementing error handling and logging
- Building command-line interfaces with argparse
- Working with external APIs and web scraping (requests, BeautifulSoup)
- Data manipulation with pandas and numpy
- Creating reusable functions and classes
- Script optimization and performance tuning

## Common Use Cases
- File system operations and batch processing
- Data extraction, transformation, and loading (ETL)
- API integrations and web automation
- System administration and DevOps tasks
- Data analysis and reporting automation
- Building tools and utilities for development workflows

## Key Principles
- Write readable, maintainable code with clear documentation
- Follow PEP 8 style guidelines
- Use virtual environments for dependency management
- Implement proper logging instead of print statements
