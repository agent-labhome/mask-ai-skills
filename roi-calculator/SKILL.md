# Skill: roi-calculator

_创建一个专业的ROI计算器，帮助企业和个人量化投资回报。

## 适用场景

- 营销活动效果评估
- 业务投资决策
- 项目可行性分析
- 客户提案价值证明

## 核心功能

1. **投资回报计算**
   - ROI = (收益 - 成本) / 成本 × 100%
   - 投资回收期计算
   - 净现值(NPV)分析

2. **多场景模板**
   - 营销活动ROI
   - 工具/软件投资
   - 培训/课程投资
   - 广告投放回报

3. **可视化输出**
   - 图表展示收益曲线
   - 敏感度分析
   - 对比分析表

## 使用方法

提供以下信息，我将为你创建定制化ROI计算器：

```
我需要一个[营销活动/软件投资/培训]的ROI计算器
- 初始投资: $____
- 预期收益: $____
- 时间周期: ____个月
- 其他成本: $____
```
