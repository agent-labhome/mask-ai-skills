# 任务管理助手

## 功能

智能任务管理工具，帮助你计划、排序、跟踪。

## 使用场景

### 1. 计划制定

**输入:** 目标 + 时间

**输出:**
- 分解任务
- 时间节点
- 里程碑

### 2. 优先级排序

**输入:** 任务列表

**输出:**
- 优先级排序
- 紧急/重要矩阵
- 时间建议

### 3. 进度跟踪

**输入:** 当前进度

**输出:**
- 完成率
- 风险预警
- 调整建议

### 4. 每日规划

**输入:** 今日任务

**输出:**
- 时间安排
- 重点任务
- 休息提醒

## 参数

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| task | string | 是 | 任务类型 |
| items | string | 是 | 任务列表 |
| deadline | string | 否 | 截止日期 |

## 示例

### 优先级排序

**输入:**
```json
{
  "task": "排序",
  "items": ["写报告","开会","回邮件","学习新技能"]
}
```

**输出:**
```
📋 优先级排序 (艾森豪威尔矩阵)

🔴 紧急且重要:
1. 写报告 (今天必须完成)

🟡 重要不紧急:
2. 学习新技能 (安排时间)

🟢 紧急不重要:
3. 回邮件 (批量处理)

⚪ 不紧急不重要:
4. 开会 (可推迟)

💡 建议:
- 上午先做报告
- 下午批量回邮件
- 每天留1小时学习
```
