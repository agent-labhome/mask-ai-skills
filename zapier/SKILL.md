# Zapier

## 功能
- 自动执行任务
- 提供专业建议
- 生成优化方案

## Core Capabilities

### Automation & Workflow Creation
- **Zap Building**: Create automated workflows (Zaps) connecting 6,000+ apps
- **Multi-Step Zaps**: Build complex workflows with conditional logic, filters, and delays
- **Trigger-Action Automation**: "When X happens in App A, do Y in App B"
- **Custom Logic**: Paths (if/then branches), filters, formatting, and data transformation
- **Webhooks**: Both trigger and action capabilities for custom integrations

### Data Handling & Transformation
- **Data Mapping**: Transform and reshape data between apps
- **Formatter Tool**: Text manipulation, date formatting, number operations
- **Lookup Tables**: Reference data for dynamic workflows
- **Code Steps**: Run JavaScript/Python for custom logic (requires Code by Zapier)
