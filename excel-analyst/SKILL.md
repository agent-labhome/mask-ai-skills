# Excel Analyst

帮助你用Excel进行数据分析。

## 功能
- 自动执行任务
- 提供专业建议
- 生成优化方案

## 核心能力

### 1. 数据处理

- 格式转换

## 使用示例

```
用户: 我想分析销售数据

助手:
## 📊 Excel数据分析指南

### 数据处理

**清洗数据:**
- 删除重复: Data → Remove Duplicates
- 填充空值: Ctrl+G → Select Blanks
- 统一格式: Text to Columns

### 关键函数

**VLOOKUP:**
```
=VLOOKUP(A2, 客户表!A:B, 2, FALSE)
```

**SUMIF:**
```
=SUMIF(A:A, "苹果", B:B)
```

**IF:**
```
=IF(B2>1000, "高", "低")
```

### 数据透视表

**创建:**
1. 选中数据
2. Insert → Pivot Table
3. 拖拽字段

**布局:**
- 行: 地区
- 列: 产品
- 值: 销售额

### 图表类型

| 数据 | 图表 |
|------|------|
| 趋势 | 折线图 |
| 对比 | 柱状图 |
| 占比 | 饼图 |
| 分布 | 散点图 |

### Dashboard模板

**要素:**
- KPI卡片
- 趋势图
- 分类对比
- 数据表格
```
