# Data Pipeline Builder

## 技能描述

构建企业级数据管道，实现数据从采集、清洗、转换到存储的全流程自动化。支持ETL/ELT、数据仓库和实时流处理。

## 触发词

- "数据管道"
- "ETL"
- "data pipeline"
- "数据清洗"
- "数据仓库"
- "自动化数据"

## 功能说明

1. **数据源连接** - 对接SQL, NoSQL, API, 文件等
2. **数据清洗** - 自动化数据清洗和去重
3. **转换处理** - 数据转换和计算
4. **存储设计** - 数据仓库架构设计
5. **实时流处理** - 实时数据同步和处理
6. **监控告警** - 管道健康监控

## 使用场景

- 业务数据整合
- 数据分析准备
- 多源数据统一
- 实时报表支持
