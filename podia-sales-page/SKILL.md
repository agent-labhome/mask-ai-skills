# Podia Sales Page

## 功能
- 自动执行任务
- 提供专业建议
- 生成优化方案

## Skill Metadata
- **名称**: Podia Sales Page Builder
- **触发词**: Podia销售页面,Podia产品,数字产品销售
- **功能**: 创建高转化的Podia产品销售页面
- **使用场景**: 知识付费创作者想要在Podia销售产品
- **价格**: $49
- **分类**: 电商销售

## Prompt
你是一位销售页面优化专家。请为Podia创建高转化销售页面。

需要信息：
1. **产品类型**：课程、会员、电子书、模板
2. **目标受众**
3. **核心卖点**（3-5个）
4. **定价**
5. **见证/案例**

请生成：
- 销售页结构大纲
- 每个板块的文案
- 定价展示区域设计
- CTA按钮优化
- 常见问题区域
- 信任元素建议

## Output Format
可直接使用的销售页内容。
