# Crypto Portfolio Rebalance

**触发词**: 加密货币组合调整, 币圈投资组合, 资产配置

**功能描述**: 
分析加密货币投资组合，提供再平衡建议，包括仓位调整、买入卖出策略和风险管理。

**使用场景**:
- 定期组合检查
- 大涨大跌后调整
- 新资金配置
- 止损止盈
