# SEO 优化助手

## 功能

专业的SEO优化工具，帮助你提升搜索引擎排名。

## 使用场景

### 1. 关键词研究

**输入:** 行业 + 目标关键词

**输出:**
- 关键词难度
- 搜索量
- 相关关键词
- 商业价值

### 2. 内容优化

**输入:** 文章内容 + 目标关键词

**输出:**
- SEO评分
- 优化建议
- 关键词密度
- 结构建议

### 3. 网站分析

**输入:** 网站URL

**输出:**
- SEO健康度
- 技术问题
- 优化建议
- 竞品对比

### 4. 竞争对手分析

**输入:** 竞争对手URL

**输出:**
- 关键词策略
- 内容策略
- 外链分析
- 流量预估

## 参数

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| task | string | 是 | 任务类型 |
| keyword | string | 是 | 关键词 |
| platform | string | 否 | 搜索引擎 |

## 示例

### 关键词研究

**输入:**
```json
{
  "task": "关键词",
  "keyword": "AI写作工具"
}
```

**输出:**
```
🔍 关键词: AI写作工具

📊 数据分析:
- 搜索量: 3,400/月
- 难度: 中等 (52)
- CPC: $2.8

💡 相关关键词:
- AI写作软件 (1,200, 中等)
- 免费AI写作 (900, 简单)
- AI写文章 (2,100, 困难)

📌 建议:
- 优先: 免费AI写作 (简单)
- 目标: AI写作软件 (中等)
- 布局: 长尾词
```

### 内容优化

**输入:**
```json
{
  "task": "优化",
  "keyword": "AI写作工具",
  "content": "这是一篇关于AI写作的文章..."
}
```

**输出:**
```
📝 内容SEO分析

✅ 得分: 72/100

👍 优点:
- 关键词出现位置好
- 段落结构清晰

⚠️ 需要改进:
- 关键词密度 1.2% (建议2-3%)
- 添加更多相关关键词
- 增加H2/H3标题

💡 具体建议:
1. 在首段增加关键词密度
2. 添加"免费AI写作"作为相关词
3. 增加小标题使用
4. 添加表格/列表增强可读性
```
