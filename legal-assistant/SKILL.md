# 法律助手

## 功能

专业的法律辅助工具，合同、法规、案例全覆盖。

## 使用场景

### 1. 合同审查

**输入:** 合同文本

**输出:**
- 风险点识别
- 条款分析
- 修改建议
- 补充条款

### 2. 法律法规

**输入:** 问题

**输出:**
- 相关法律
- 条款解读
- 适用情况
- 注意事项

### 3. 案例分析

**输入:** 案例描述

**输出:**
- 案由分析
- 判决要点
- 类似案例
- 建议

### 4. 合规检查

**输入:** 企业业务

**输出:**
- 合规风险
- 需要的资质
- 建议措施
- 相关法规

## 参数

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| task | string | 是 | 任务类型 |
| question | string | 是 | 问题 |
| jurisdiction | string | 否 | 地区 |

## 示例

### 合同审查

**输入:**
```json
{
  "task": "审查",
  "contract": "甲方向乙方提供...服务..."
}
```

**输出:**
```
📋 合同审查报告

⚠️ 风险点:
1. 违约责任不明确
2. 知识产权归属模糊
3. 争议解决条款缺失

✅ 良好条款:
- 付款方式清晰
- 服务范围明确

💡 建议:
1. 添加违约金条款
2. 明确IP归属
3. 增加仲裁条款
```
