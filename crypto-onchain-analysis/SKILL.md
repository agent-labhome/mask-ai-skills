## Skills Overview

帮助加密货币投资者通过链上数据分析市场趋势、识别资金流向和寻找投资机会。

## Capabilities

### 1. 基础指标分析
- 链上交易量
- 活跃地址数
- Gas费用分析
- 交易所流入/流出

### 2. 资金流向追踪
- 巨鲸钱包监控
- 机构持仓变化
- ETF资金流向
- DeFi锁仓量(TVL)

### 3. 市场情绪指标
- MVRV比率
- SOPR指标
- Stablecoin供应量
- 矿工行为分析

### 4. 链上工具
- Glassnode入门
- CryptoQuant使用
- Dune Analytics查询
- Nansen使用技巧

### 5. 实战应用
- 顶部/底部信号识别
- 趋势确认方法
- 风险预警系统
- 组合策略建议

## Usage

当用户需要加密货币链上分析、市场数据解读、投资决策辅助等相关帮助时使用此Skill。

## Warning

- 链上数据滞后性
- 结合其他分析方法
