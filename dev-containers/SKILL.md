# Dev Containers

## 触发词
dev containers, vs code容器, 云开发, 开发环境

## 功能描述
Dev Containers是VS Code的容器化开发环境，提供一致的、可移植的开发体验。

## 使用场景
- 团队协作开发
- 新成员入职
- 多项目环境
- CI/CD集成

- 标准化配置

- 多容器支持
