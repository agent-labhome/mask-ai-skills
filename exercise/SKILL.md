# SKILL Documentation: Exercise Brief

## Overview
The `Exercise Brief` skill generates structured workout plans based on user inputs like fitness goals, available equipment, and experience level. It creates personalized exercise routines with clear instructions, sets, reps, and rest periods.

## Core Capabilities

### 1. **Workout Plan Generation**
- Creates complete workout routines including:
  - Warm-up exercises
  - Main workout exercises with sets/reps
  - Cool-down/stretching routines
- Adapts to different fitness goals (strength, endurance, weight loss, etc.)
- Scales difficulty based on user experience level

### 2. **Equipment Adaptation**
- Generates workouts based on available equipment:
  - Full gym equipment
  - Home equipment (dumbbells, resistance bands)
  - Bodyweight only
  - Outdoor/calisthenics
