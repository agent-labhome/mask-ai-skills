# Amazon FBA Research - Amazon FBA选品研究

## Skill Metadata
- **名称**: Amazon FBA Research
- **触发词**: amazon fba, fba选品,亚马逊选品
- **功能**: 分析Amazon FBA市场数据，提供选品建议和竞争分析
- **使用场景**: 亚马逊卖家、FBA入门、跨境电商选品
- **价格**: $49/$99/$199

## 功能说明

### 核心能力
1. **市场分析** - 分析类目容量和增长趋势
2. **竞品研究** - 分析竞争对手销量和定价
3. **利润计算** - 计算FBA费用和净利润
4. **选品建议** - 提供低竞争高利润产品建议

### 使用指南
- 指定目标市场（US/UK/DE等）
- 说明预算和品类偏好
- 获取选品报告

###示例
```
用户: 在美国市场找家居类FBA产品
输出: 3个推荐品类、预计利润、竞争分析
```
