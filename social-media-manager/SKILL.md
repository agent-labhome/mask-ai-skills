# 社交媒体运营助手

## 功能

一站式社交媒体运营工具，帮助你规划、内容、发布、分析。

## 使用场景

### 1. 内容规划

**输入:** 目标平台 + 行业

**输出:**
- 内容日历
- 热点话题
- 发布频率建议

### 2. 帖子生成

**输入:** 主题 + 平台

**输出:**
- 适配的文案
- Hashtags
- 发布时间建议

### 3. 数据分析

**输入:** 帖子数据

**输出:**
- 表现分析
- 优化建议
- 粉丝画像

### 4. 竞品分析

**输入:** 竞品账号

**输出:**
- 内容策略
- 互动分析
- 增长建议

## 参数

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| task | string | 是 | 任务类型 |
| platform | string | 是 | 平台 |
| topic | string | 否 | 主题 |

## 示例

### 内容规划

**输入:**
```json
{
  "task": "规划",
  "platform": "小红书",
  "industry": "美妆"
}
```

**输出:**
```
📅 本周内容日历

周一: 产品种草
  标题: xxx
  时间: 20:00

周二: 教程分享
  标题: xxx
  时间: 12:00

...

💡 热点话题:
#情人节 #换季护肤
```

### 帖子生成

**输入:**
```json
{
  "task": "写帖",
  "platform": "Twitter",
  "topic": "AI工具推荐"
}
```

**输出:**
```
🧵 7个AI工具让你的效率翻倍

1. ChatGPT - 写作助手
2. Midjourney - AI绘图
3. Notion - 笔记整理

点击链接看详细→

#AI #工具 #效率
```
