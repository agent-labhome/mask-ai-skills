# Podcast Clip Generator

## 功能
- 自动执行任务
- 提供专业建议
- 生成优化方案

## Skill Description
播客片段生成器 - 将长播客音频转换为适合社交媒体传播的短视频片段。支持多平台输出（TikTok、Instagram Reels、YouTube Shorts）。

## Use Cases
- 将播客亮点提取为15-60秒短视频
- 生成带字幕的社交媒体片段
- 自动识别精彩语录并剪辑

## Instructions

### Input Requirements
- 播客音频文件或YouTube链接
- 目标平台（TikTok/Instagram/YouTube）
- 期望片段时长（15s/30s/60s）
- 主题/关键词（可选）

### Output
- 视频文件（MP4）
- 包含自动字幕
- 适配各平台尺寸

### Process
1. 提取音频或下载视频
2. 使用语音识别生成文字稿
3. 识别高能量/金句片段
4. 裁剪并添加字幕
5. 导出适配平台的视频

## Tools Used
- yt-dlp: 下载音视频
- whisper: 语音识别
- ffmpeg: 视频处理
- 文字转语音TTS（可选）

## Customization
- 字幕样式可定制
- 支持多语言
- 可添加品牌水印


## 使用场景
本技能适用于需要帮助的用户，可应用于多种工作场景。
