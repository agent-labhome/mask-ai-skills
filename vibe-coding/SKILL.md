# Vibe Coding

## 功能
- 自动执行任务
- 提供专业建议
- 生成优化方案

## 概念

"Vibe Coding" = 配合AI写代码
- 不需要懂所有技术细节
- 用自然语言描述需求
- AI生成大部分代码
- 人类只做审查和整合

## 核心原则

1. **描述结果，而非过程**
   - ❌ "写一个for循环"
   - ✅ "把列表里的价格加起来"

2. **相信AI能理解**
   - 给足够上下文
   - 描述你想要什么，而不是怎么实现

3. **快速迭代**
   - 先让AI生成
   - 不满意 → 重新生成
   - 而不是自己改代码

## 常用工具

| 工具 | 用途 |
|------|------|
| Cursor | AI代码编辑器 |
| GitHub Copilot | 代码补全 |
| Claude Code | Agent编程 |
| v0 | UI生成 |
| Bolt.new | 全栈生成 |

## 工作流

1. 描述你想要的应用
2. AI生成初始代码
3. 运行测试
4. 描述需要修改的地方
5. AI修改
6. 重复直到满意

## 适合谁

- 创业者快速MVP
- 设计师转开发
- 产品经理验证想法
- 任何人想快速实现想法


## 使用场景
本技能适用于需要帮助的用户，可应用于多种工作场景。
