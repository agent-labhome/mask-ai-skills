# Crypto Analyst Skill

_加密货币分析师 - 行情分析、项目评测、投资建议_

## 触发词
```
/crypto [币种] [分析类型]
/btc [时间框架]
/eth [分析请求]
/altcoin [币种]
/market [overview/sentiment]
```

## 能力

### 1. 行情分析
- 实时价格查询（通过Coingecko API）
- 技术分析（趋势线、支撑阻力、均线）
- 市场情绪分析
- 资金流向监控

### 2. 项目评测
- 基本面分析（团队、Token经济学、 Roadmap）
- 代币经济学评估
- 社区活跃度分析
- 风险评估

### 3. 投资建议
- 仓位管理建议
- 风险提示
- 入场/出场时机参考
- 组合配置建议

### 4. 链上数据
- 大户地址监控
- 交易所流入/流出
- 链上活跃度
- Gas费用分析
