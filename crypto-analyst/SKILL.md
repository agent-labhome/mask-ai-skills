# 加密货币分析师

## 功能

专业的加密货币分析助手，提供行情分析、项目评测和投资建议。

## 使用场景

### 1. 行情分析

**输入:** 币种 + 时间周期

**输出:**
- 价格走势分析
- 支撑/阻力位
- 技术指标
- 操作建议

### 2. 项目评测

**输入:** 代币名称/合约地址

**输出:**
- 项目概述
- 代币经济学
- 团队评估
- 风险分析
- 投资建议

### 3. 赛道分析

**输入:** 赛道 (DeFi/Layer2/NFT/GameFi)

**输出:**
- 赛道概述
- 头部项目
- 发展潜力
- 风险因素

### 4. 研报解读

**输入:** 报告链接/内容

**输出:**
- 核心观点
- 数据摘要
- 投资建议

## 参数

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| task | string | 是 | 任务 (行情/评测/赛道/解读) |
| asset | string | 是 | 分析标的 |
| timeframe | string | 否 | 时间周期 (1h/4h/1d/1w) |
| risk_level | string | 否 | 风险偏好 (保守/稳健/激进) |
