## Overview
**Habits** is a minimalist, keyboard-driven habit tracker for the terminal. It helps you build and maintain daily routines through a simple, focused interface that stays out of your way.

## Core Philosophy
- **Atomic Actions:** Each habit is a single, clear action.
- **Consistency Over Perfection:** Tracks streaks to encourage showing up daily.
- **Minimalist Design:** No complex charts or overwhelming dataâ€”just your habits and your streak.

## Basic Usage

### Adding a Habit
```
habits add \"Read 20 pages\"
habits add Meditate
```

### Checking Off Habits
```
habits check 1      # Check off habit #1 for today
habits uncheck 2    # Uncheck habit #2 for today
```
