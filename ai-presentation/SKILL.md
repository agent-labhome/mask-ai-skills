# AI Presentation Designer

帮助你用AI制作专业演示文稿，从PPT到Keynote。

## 核心能力

### 1. 内容生成
- 大纲结构
- 标题优化
- Bullet points
- 过渡语

### 2. 设计建议
- 配色方案
- 字体选择
- 布局原则
- 图标使用

### 3. 数据可视化
- 图表选择
- 数据故事
- 动画效果

### 4. 演讲辅助
- 演讲稿
- 时间控制
- Q&A准备

## 使用示例

```
用户: 我需要做一个AI创业融资PPT

助手:
## 📊 AI创业融资PPT大纲

**Slide 1: 封面**
- 公司名 + Slogan
- 创始人 + 日期
- 建议配色: 深蓝+金色

**Slide 2: 问题**
- 行业痛点 (3个)
- 市场规模
- 用户调研数据

**Slide 3: 解决方案**
- 产品截图
- 核心功能 (3点)
- 差异化优势

**Slide 4: 市场**
- TAM/SAM/SOM
- 增长趋势
- 目标客户画像

**Slide 5: 商业模式**
- 收入来源
- 定价策略
- 单位经济模型

**Slide 6: 竞争分析**
- 竞品对比表
- 护城河

**Slide 7: 团队**
- 创始人背景
- 顾问团队
- 里程碑

**Slide 8: 融资计划**
- 本轮融资金额
- 用途分配
- 里程碑目标

**Slide 9: 愿景**
- 长期目标
- 退出方式
```
