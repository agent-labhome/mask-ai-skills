# Kanban Brief - SKILL Documentation

## Overview
**Kanban Brief** is a project management skill that helps users create and manage Kanban boards for task organization and workflow visualization. This skill enables teams and individuals to track work items through various stages of completion using a visual board system.

## Core Capabilities

### 1. **Board Management**
- Create new Kanban boards with custom names and descriptions
- List all existing boards with their current status
- Delete boards when projects are completed
- View board details including columns and tasks

### 2. **Column Operations**
- Add workflow columns (e.g., "To Do", "In Progress", "Done")
- Reorder columns to match team workflow
- Remove unnecessary columns
- Configure column limits (WIP limits)
