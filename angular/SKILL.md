## 功能
- 自动执行任务
- 提供专业建议
- 生成优化方案

## Core Competencies
- **Framework**: Angular (v2+), TypeScript
- **Architecture**: Component-based, modules, services, dependency injection
- **Templates**: HTML with Angular directives, reactive forms, template-driven forms
- **State Management**: RxJS observables, services, NgRx (optional)
- **Routing**: Angular Router with lazy loading

## Key Concepts
- Components & data binding
- Directives (structural/attribute)
- Pipes for data transformation
- HTTP client with interceptors
- Angular CLI for project management
- Testing with Jasmine/Karma

## Common Patterns
- Smart/dumb component architecture
- Reactive programming with RxJS
- Form validation (reactive/template)
- Route guards and resolvers
- Custom pipes/directives
