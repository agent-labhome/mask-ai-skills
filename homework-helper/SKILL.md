### 1. 题目解答

**输入:** 题目

**输出:**

**输出:**

### 3. 知识讲解

## 示例

### 题目解答

**输入:**
```json
{
  "task": "解答",
  "subject": "数学",
  "question": "求函数f(x)=x^2+2x+1的导数"
}
```

**输出:**
```
📝 解答:

f(x) = x² + 2x + 1

使用求导公式:
- d/dx(xⁿ) = n·xⁿ⁻¹
- d/dx(ax) = a
- d/dx(c) = 0

求导:
f'(x) = 2x + 2

📌 知识点:
1. 幂函数求导
2. 导数的线性组合

💡 技巧:
这个函数可以写成 (x+1)²
导数 2(x+1) = 2x + 2
```
