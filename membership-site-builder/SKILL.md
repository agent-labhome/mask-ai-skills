# Membership Site Builder

## 技能描述

构建付费会员网站，创建专属内容社区。包含会员体系、付费墙、社区功能和持续变现系统。

- "会员网站"

## 功能说明

1. **会员体系** - 分层会员设计
2. **内容保护** - 付费墙设置
3. **社区功能** - 论坛/讨论区
4. **付费订阅** - 月付/年付/终身
5. **会员管理** - 用户管理和数据分析
6. **自动化的** - 自动续费、到期提醒

## 使用场景

- 专业课程会员
- 行业社群
- 内容订阅
- 专家会员
