# 冷邮件营销活动

## 功能

创建高回复率的冷邮件序列，用于B2B销售。

## 使用场景

### 1. 潜在客户开发
- 首封开发信
- 跟进邮件
- 价值传递

### 2. 活动邀请
-  webinar邀请
- 活动推广
- 峰会邀约

### 3. 合作洽谈
- 合作提案
- 联盟邀请
- 资源对接

## 参数

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| campaign_type | string | 是 | 活动类型 (outreach/invitation/partnership) |
| sequence_length | number | 否 | 序列长度 默认3 |
| target_industry | string | 是 | 目标行业 |
