## 功能描述
Cursor Rules是优化Cursor AI编程的系统：
- 自定义规则文件
- 代码风格规范化
- 项目上下文理解
- 团队协作统一
- 效率大幅提升

## 核心规则类型
1. **项目规则** - 整体架构约束
2. **语言规则** - 特定语言规范
3. **框架规则** - 框架最佳实践
4. **测试规则** - 测试要求
5. **文档规则** - 文档标准

## 创建方法
```
创建 .cursorrules 文件

```
