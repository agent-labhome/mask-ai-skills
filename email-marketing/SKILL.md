# 邮件营销助手

## 功能

专业的邮件营销工具，帮你写模板、自动化、分析效果。

## 使用场景

### 1. 邮件模板

**输入:** 场景 + 目标

**输出:**
- 标题优化
- 正文模板
- CTA设计

### 2. 自动化流程

**输入:** 用户行为

**输出:**
- 触发条件
- 流程设计
- 时间安排

### 3. 回复处理

**输入:** 用户回复

**输出:**
- 智能分类
- 回复建议
- 跟进策略

### 4. 效果分析

**输入:** 邮件数据

**输出:**
- 打开率分析
- 点击率分析
- 优化建议

## 参数

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| task | string | 是 | 任务类型 |
| scenario | string | 是 | 应用场景 |
| audience | string | 否 | 目标用户 |

## 示例

### 邮件模板

**输入:**
```json
{
  "task": "模板",
  "scenario": "新产品发布",
  "audience": "现有客户"
}
```

**输出:**
