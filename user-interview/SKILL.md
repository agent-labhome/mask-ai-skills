帮助你进行有效的用户访谈。

### 1. 访谈准备
- 目标设定
- 用户筛选
- 问卷设计
- 流程规划

### 2. 访谈技巧
- 开场破冰
- 提问技巧
- 倾听引导
- 追问深挖

### 3. 记录分析
- 录音转写

### 4. 行动转化

- 优先级

- 跟进计划

## 使用示例

```
用户: 我想做用户访谈了解需求

助手:
## 🎤 用户访谈指南

### 访谈准备

**确定目标:**
- 了解用户痛点
- 验证产品假设
- 发现新需求

**筛选用户:**
- 目标用户画像
- 使用过竞品
- 愿意配合

**访谈提纲:**
```
1. 自我介绍 (2min)
2. 背景了解 (5min)
3. 使用场景 (10min)
4. 痛点挖掘 (15min)
5. 现有方案 (10min)
6. 产品反馈 (10min)
7. 总结 (5min)
总计: 约57分钟
```

### 提问技巧

**开放问题:**
- "请描述一下..."
- "能具体说说吗?"

**追问:**
- "为什么?"
- "后来呢?"
- "能举个例子吗?"

**避免:**
- 引导性问题
- 假设性问题
- 封闭问题

### 记录要点

**必须记录:**
- 用户原话
- 情绪/表情
- 关键痛点
- 意外发现

### 常见问题归类

1. 功能需求
2. 使用体验
3. 价格敏感
4. 信任问题
5. 竞品对比
```
