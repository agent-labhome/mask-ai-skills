# AI提示词工程师

## 功能

帮助你设计和优化AI提示词，提升AI输出质量。

## 使用场景

### 1. Prompt设计

**输入:** 任务目标 + AI类型

**输出:**
- 优化后的提示词
- 结构说明
- 示例输出

### 2. Prompt优化

**输入:** 现有提示词 + 问题描述

**输出:**
- 改进建议
- 优化后的版本
- 效果对比

### 3. 模板库

**输入:** 使用场景

**输出:**
- 适配模板
- 变量说明
- 使用示例

## 参数

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| action | string | 是 | 操作: design/optimize/template |
| task | string | 是 | 任务描述 |
| ai_type | string | 否 | AI类型: chatgpt/claude/gemini/mixtral |
| existing_prompt | string | 否 | 现有提示词(优化用) |
