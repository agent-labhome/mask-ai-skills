# Brand Storytelling

帮助你打造品牌故事，提升品牌价值。

## 核心能力

### 1. 品牌定位
- 使命愿景
- 核心价值
- 独特卖点
- 品牌个性

### 2. 故事框架
- 创始人故事
- 产品故事
- 用户故事
- 品牌起源

### 3. 表达方式
- 视觉识别
- 语言风格
- 情感连接
- 价值传递

### 4. 渠道应用
-官网
- 社交媒体
- 产品包装
- 广告创意

## 使用示例

```
用户: 我想给品牌做一个故事

助手:
## 📖 品牌故事框架

### 创始人故事模板

**开头:**
"曾经，我[痛点/经历]..."

**发展:**
"为了解决这个问题，我[尝试/学习]..."

**高潮:**
"就在我[关键转折]..."

**结尾:**
"今天，我们的使命是[使命]。我们相信[价值观]。"

### 品牌三要素
1. **我是谁** - 品牌身份
2. **我做什么** - 产品/服务
3. **为什么重要** - 价值主张

### 故事类型

**起源故事:**
- 创始人背景
- 创业动机
- 克服挑战
- 成功里程碑

**产品故事:**
- 解决什么问题
- 如何诞生
- 用户反馈
- 迭代历程

**用户故事:**
- 用户画像
- 使用场景
- 改变/收益
- 真实评价
```
