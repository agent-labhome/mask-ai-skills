# Gumroad Seller

帮助你优化Gumroad销售，从产品到转化全流程。

## 功能
- 自动执行任务
- 提供专业建议
- 生成优化方案

### 1. 产品定位
- 目标市场分析
- 差异化定位
- 定价策略
- 捆绑销售

### 2. Landing Page优化
- 标题撰写
- 描述结构
- 图片设计
- CTA优化

### 3. 销售文案
- 痛点挖掘
- 解决方案
- 社会证明
- 稀缺感营造

### 4. 增长策略
- 邮件营销
- 社交推广
- 联盟分销
- SEO优化

## 使用示例

```
用户: 我卖AI提示词模板，怎么提高转化？

助手:
## 💰 AI提示词Gumroad优化指南

### 定价建议
- 单个模板: $9-19
- 捆绑包(10+): $29-49
- 高级版: $99+

### Landing Page结构
1. **主标题**: "[职业]的[数量]个AI提示词，节省[时间]"
2. **副标题**: 解决[具体痛点]
3. **预览**: 展示3-5个最值钱的模板
4. **评价**: 3-5个真实买家评价
5. **价格**: 突出原价vs现价

### 转化技巧
- 限时优惠 (48小时)
- 买一送一 (相关模板)
- 评论区活跃回复
- 免费升级承诺

### 捆绑策略
- 基础包: $19
- 全家桶: $39 (+$20但多5倍内容)
- "最划算"标签
```
