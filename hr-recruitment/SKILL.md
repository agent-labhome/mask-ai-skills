## HR招聘助手

### 功能

专业的HR招聘工具，简历、面试、JD全覆盖。

### 使用场景

#### 1. 简历筛选

**输入:** 简历 + 职位要求

**输出:**
- 匹配度评分
- 关键亮点
- 风险点
- 建议

#### 2. 面试题库

**输入:** 职位 + 轮次

**输出:**
- 技术问题
- 行为问题
- 压力测试
- 评分标准

#### 3. JD撰写

**输入:** 职位 + 要求

**输出:**
- 职位描述
- 任职要求
- 薪资范围
- 发展路径

#### 4. 薪资参考

**输入:** 职位 + 地区

**输出:**
- 薪资范围
- 福利建议
- 竞争分析

### 参数

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| task | string | 是 | 任务类型 |
| position | string | 是 | 职位 |
| level | string | 否 | 级别 |
