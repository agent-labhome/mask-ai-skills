## Core Concept
A decentralized digital currency operating on a peer-to-peer network without central authority.

## Key Features
- **Decentralized**: No central bank or administrator
- **Limited Supply**: Capped at 21 million BTC
- **Blockchain**: Public ledger recording all transactions
- **Proof-of-Work**: Mining secures the network and processes transactions

## Primary Use Cases
1. Digital store of value (\"digital gold\")
2. Peer-to-peer electronic cash system
3. Hedge against inflation/fiat currency devaluation
4. Settlement layer for value transfer

## Essential Knowledge
- Private keys control ownership
- Transactions are irreversible
- Network secured by cryptographic proof
- Approximately 10-minute block times

- `bitcoin-cli` - Command-line interface
