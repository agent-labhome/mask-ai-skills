## Overview
Bitcoin is the first decentralized cryptocurrency, introduced in a 2008 whitepaper by Satoshi Nakamoto. It enables peer-to-peer transactions without intermediaries, secured by cryptographic proof and a distributed timestamp server (blockchain). Bitcoin's primary value proposition is as \"digital gold\"â€”a censorship-resistant store of value with predictable, transparent monetary policy.

## Core Technology
- **Blockchain**: Public ledger of all transactions, maintained by nodes
- **SHA-256**: Cryptographic hash function used for mining
- **UTXO Model**: Unspent Transaction Output model for tracking ownership
- **Difficulty Adjustment**: Every 2016 blocks (~2 weeks) to maintain 10-minute block time
- **Halving**: Block reward reduces 50% every 210,000 blocks (~4 years)
