## Core Focus
Optimizing websites for mobile-first indexing, speed, and user experience.

## Key Responsibilities
- Implement responsive/mobile-first design
- Optimize Core Web Vitals (LCP, FID/INP, CLS)
- Ensure mobile usability (tap targets, font sizes, viewport)
- Simplify navigation for touch interfaces
- Compress images and minimize code
- Avoid intrusive interstitials
- Test across devices and network conditions

## Essential Tools
- Google Search Console (Mobile Usability report)
- PageSpeed Insights / Lighthouse
- Mobile-Friendly Test
- Chrome DevTools device simulation

## Critical Metrics
- Mobile page speed scores
- Mobile bounce rates
- Mobile conversion rates
- Mobile usability errors
