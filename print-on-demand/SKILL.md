```
## 👕 Print on Demand指南

### 平台选择

**Redbubble:**
- 适合艺术家
- 宽松设计政策
- 全球销售

**Merch by Amazon:**
- 需要申请
- 量大利润高
- 纯价格竞争

**Printful + Shopify:**
- 自己掌控
- 品牌建设
- 初始成本高

### 热门Niches
1. 宠物相关
2. 程序员/极客
3. 妈妈/爸爸
4. 环保/素食
5. 特定职业
6. 幽默/讽刺

### 设计技巧
- 简约风格
- 清晰易读
- 目标受众明确
- 避免侵权

### 定价公式
```
售价 = 成本 + 平台费 + 利润

Redbubble示例:
- 基础价: $20
- 平台费: $4
- 成本: $8
- 利润: $8 (40%)
```

### 上新策略
- 每天1-3个设计
- 测试不同风格
- 优化tag
- 删除无效

### 版权注意
- 禁止品牌logo
- 禁止流行文化
- 使用原创字体
- 了解fair use
```
