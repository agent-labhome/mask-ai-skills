# Translator

## 功能

AI翻译助手，支持多语言翻译、专业术语处理、本地化优化。

## 使用场景

### 1. 文档翻译

**输入:** 原文 + 目标语言

**输出:**
- 翻译后的文档
- 保持原文格式
- 专业术语翻译说明

### 2. 专业术语翻译

**输入:** 领域 + 术语

**输出:**
- 标准翻译
- 行业常用表达
- 上下文用法示例

### 3. 本地化优化

**输入:** 原文 + 目标市场

**输出:**
- 文化适配版本
- 当地表达习惯
- SEO本地化建议

### 4. 电商产品描述

**输入:** 产品信息 + 目标国家

**输出:**
- 当地化产品描述
- 关键词优化
- 文化敏感点检查
