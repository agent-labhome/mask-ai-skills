# 智能客服助手

## 功能

AI驱动的智能客服系统，自动回复、分类、分析。

## 使用场景

### 1. 自动回复

**输入:** 用户问题

**输出:**
- 智能回答
- 相关知识
- 升级建议

### 2. 问题分类

**输入:** 用户消息

**输出:**
- 问题类型
- 紧急程度
- 处理建议

### 3. 情绪分析

**输入:** 用户消息

**输出:**
- 情绪判断
- 风险预警
- 应对策略

### 4. 知识库

**输入:** 产品信息

**输出:**
- FAQ生成
- 回答模板
- 知识结构

## 参数

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| task | string | 是 | 任务类型 |
| message | string | 是 | 用户消息 |
| context | string | 否 | 上下文 |

## 示例

### 自动回复

**输入:**
```json
{
  "task": "回复",
  "message": "这个产品怎么使用？"
}
```

**输出:**
```
您好！感谢咨询~

我们的产品使用非常简便：

1. 下载App并注册
2. 绑定您的设备
3. 按照引导完成设置

如有其他问题，随时问我！

📞 客服热线: xxx
```

### 情绪分析

**输入:**
```json
{
  "task": "情绪",
  "message": "太垃圾了！完全不能用！"
}
```

**输出:**
```
😠 情绪: 愤怒 (高风险)

⚠️ 预警: 可能投诉/差评

💡 建议:
1. 立即升级人工处理
2. 态度要真诚道歉
3. 提供补偿方案
4. 记录并跟进

🚨 建议立即响应！
```
