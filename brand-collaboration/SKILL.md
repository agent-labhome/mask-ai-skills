# Brand Collaboration

帮助你管理品牌合作和KOL营销。

## 功能
- 自动执行任务
- 提供专业建议
- 生成优化方案

## 核心能力

### 1. 合作类型
- KOL/KOC合作
- 品牌联名
- 资源置换
- 佣金分成

### 2. 合作流程
- 寻找达人
- 联系沟通
- 合作谈判
- 合同签订

### 3. 合作管理
- 进度跟进
- 内容审核
- 效果追踪
- 关系维护

### 4. 报价策略
- 报价公式
- 阶梯价格
- 捆绑优惠
- 长期合作

## 使用示例

```
用户: 我想找小红书达人合作

助手:
## 🤝 品牌合作指南

### 达人筛选标准
- 粉丝: 1K-10K (KOC)
- 互动率: >5%
- 内容质量
- 粉丝画像

### 联系模板
```
Hi [名字],

我们是[品牌名]，看到你的[内容类型]很棒！

合作形式:
- 产品寄送 + 付费推广
- 或纯佣金分成

感兴趣的话，可以回复了解详情~

[你的名字]
```

### 报价参考 (小红书)
| 粉丝量 | 图文 | 视频 |
|--------|------|------|
| 1K-5K | 100-300 | 200-500 |
| 5K-10K | 300-800 | 500-1500 |
| 10K-50K | 800-3000 | 1500-5000 |
| 50K+ | 3000+ | 5000+ |

### 合同要点
- 交付时间
- 内容修改次数
- 发布时间
- 效果数据
- 独家期限

### 效果追踪
- 阅读/播放量
- 互动数
- 转化数
- ROI计算
```
