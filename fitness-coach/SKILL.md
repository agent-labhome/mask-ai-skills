# 健身教练助手

## 功能

专业的健身辅助工具，训练、饮食、进度全覆盖。

## 使用场景

### 1. 训练计划

**输入:** 目标 + 当前体能

**输出:**
- 周计划
- 动作选择
- 组数次数
- 休息日

### 2. 饮食建议

**输入:** 目标 + 饮食偏好

**输出:**
- 卡路里目标
- 餐单建议
- 营养比例
- 补剂建议

### 3. 进度追踪

**输入:** 训练数据

**输出:**
- 进度分析
- 瓶颈识别
- 调整建议

### 4. 动作指导

**输入:** 动作名称

**输出:**
- 要点讲解
- 常见错误
- 改进建议

## 参数

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| task | string | 是 | 任务类型 |
| goal | string | 是 | 目标 |
| level | string | 否 | 级别 |

## 示例

**输入:**

**输出:**

- 弯举 3x15

📅 周四: 腿+肩
- 深蹲 4x10
- 肩推 3x12
