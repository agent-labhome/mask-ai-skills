# v0 AI UI 生成器

v0 是 Vercel 推出的 AI 驱动的 UI 生成工具，基于 Shadcn UI 和 Tailwind CSS。

## 核心功能

- **文本转 UI**: 描述生成 React 组件
- **Shadcn UI 集成**: 生成现代化 React UI
- **Tailwind CSS**: 响应式样式
- **代码可定制**: 完整源代码输出

## 使用方式

1. 描述你想要 UI（如"一个现代的登录表单"）
2. AI 生成多个版本供选择
3. 点击复制或进一步编辑
4. 集成到你的项目中

## 适用场景

- 快速原型 UI
- 学习现代 React 开发
- 生成 Shadcn UI 组件
- MVP 界面搭建

## 技术栈

- React + TypeScript
- Tailwind CSS
- Shadcn UI
- Radix UI primitives

## 最佳实践

1. 描述具体化（"一个深色主题的侧边栏导航"）
2. 指定设计系统（"使用 Shadcn UI 风格"）
3. 迭代调整直到满意
4. 复制代码后根据需求微调

## 对比类似工具

- Bolt.new: 全栈应用生成
- Lovable: 完整 Web 应用
