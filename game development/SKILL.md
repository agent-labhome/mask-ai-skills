## Core Technical Skills

### **Programming Languages**
- **C++**: Industry standard for AAA engines (Unreal, custom engines)
- **C#**: Primary language for Unity development
- **Python**: Tools development, automation, scripting
- **JavaScript/TypeScript**: Web games, some mobile frameworks
- **GLSL/HLSL**: Shader programming

### **Game Engines & Frameworks**
- **Unity**: C# scripting, component system, asset pipeline
- **Unreal Engine**: Blueprints visual scripting, C++ gameplay programming
- **Godot**: GDScript, node-based architecture
- **Custom Engines**: OpenGL/DirectX/Vulkan, entity-component systems

### **Specialized Development Areas**

#### **Gameplay Programming**
- Character controllers, AI behaviors, combat systems
- Game mechanics implementation
- Physics integration and custom systems
- Input handling across platforms
