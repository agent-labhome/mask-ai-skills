## Core Competencies
- **Infrastructure as a Service (IaaS)**: Virtual machines, storage, networking
- **Platform as a Service (PaaS)**: Managed databases, app runtimes, serverless
- **Software as a Service (SaaS)**: Cloud applications and collaboration tools

## Key Providers
- AWS (Amazon Web Services)
- Microsoft Azure
- Google Cloud Platform (GCP)

## Essential Concepts
- **Virtualization**: Abstraction of physical resources
- **Scalability**: Auto-scaling, load balancing
- **Resiliency**: High availability, disaster recovery
- **Security**: IAM, encryption, compliance
- **Cost Management**: Pay-per-use, resource optimization

## Common Services
- Compute (EC2, VMs, Lambda)
- Storage (S3, Blob Storage)
- Databases (RDS, Cosmos DB)
- Networking (VPC, CDN, DNS)
