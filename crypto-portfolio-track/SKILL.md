# Crypto Portfolio Track - 加密货币组合追踪

## Skill Metadata
- **名称**: Crypto Portfolio Track
- **触发词**: 加密货币组合, crypto portfolio, 币圈
- **功能**: 分析和追踪加密货币投资组合，提供调仓建议
- **使用场景**: 加密投资、风险管理、资产配置
- **价格**: $49/$99/$199

## 功能说明

### 核心能力
1. **组合分析** - 分析当前持仓和盈亏
2. **风险评估** - 评估组合风险暴露
3. **调仓建议** - 提供买卖和再平衡建议
4. **市场监控** - 监控大户地址和链上数据

### 使用指南
- 提供持仓信息或钱包地址
- 说明投资目标和风险偏好
- 获取分析报告

### 示例
```
用户: 分析我的BTC+ETH+SOL组合
输出: 持仓分析、风险评估、调仓建议
```
