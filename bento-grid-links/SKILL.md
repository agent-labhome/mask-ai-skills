# Bento Grid Links

## 触发词
bento grid, bento.me, link in bio, 链接页面

## 功能描述
Bento Grid是现代化的链接聚合工具，以网格布局展示内容和社交链接。

## 使用场景
- 社交媒体Bio链接
- 作品集展示
- 产品/服务介绍
- 活动和事件页面

## 核心功能
- 网格布局系统
- 链接和社交整合
- 作品展示
- 自定义主题
- 分析统计
- 移动端优化
