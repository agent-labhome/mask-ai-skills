# AI Video Short Generator

_Create viral short-form videos (TikTok, Reels, YouTube Shorts) with AI_

## When to Use

Use this skill when you need to:
- Generate short videos from text/prompts
- Create faceless content videos
- Auto-edit and add subtitles
- Repurpose long videos into shorts
- Add voiceovers to videos
- Create video thumbnails
- Batch produce short-form content

## Workflow

### 1. Concept & Script
- Define topic and hook
- Write script (60-90 seconds)
- Add captions and emojis
- Plan visual elements

### 2. AI Generation
- Choose platform (InVideo, Pictory, etc.)
- Input script or article URL
- Select template and style
- Generate video

### 3. Editing & Enhancement
- Adjust timing and transitions
- Add background music
- Enhance captions
- Fix any AI artifacts

### 4. Export & Publish
- Export in correct format (9:16 for TikTok/Reels)
- Add hashtags and descriptions
- Schedule posting

1. **Educational** - Tips, how-tos, facts
2. **Entertainment** - Stories, memes, reactions

## Best Practices

- Hook in first 3 seconds
- Add captions (80% watch muted)
- Use trending sounds strategically
- Post consistently (daily)
- Engage in first hour
- Repurpose across platforms
