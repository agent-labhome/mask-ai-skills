## What is Bootstrap?

Bootstrap is an open-source CSS framework that provides responsive grid systems, pre-built components, and JavaScript plugins to accelerate web development. Originally created by Twitter, it has become one of the most popular frontend frameworks worldwide.

## Quick Start

### Installation Options

**1. CDN (Quickest for prototyping)**
```html
<!-- CSS -->
<link href=\"https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/css/bootstrap.min.css\" rel=\"stylesheet\">

<!-- JavaScript Bundle (Popper included) -->
<script src=\"https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/js/bootstrap.bundle.min.js\"></script>
```

**2. npm**
```bash
npm install bootstrap
```

**3. Download compiled files**
- Download from [getbootstrap.com](https://getbootstrap.com/docs/5.3/getting-started/download/)
