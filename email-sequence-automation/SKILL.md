# 自动化邮件序列

## 功能

创建高效的邮件自动化序列，提升转化率。

## 使用场景

### 1. 新订阅欢迎序列
- 欢迎邮件
- 价值传递
- 首次优惠

### 2. 销售漏斗
- 线索培育
- 促销通知
- 弃购召回

### 3. 会员维护
- 定期简报
- 产品更新
- 忠诚度奖励

| 参数 | 类型 | 必填 | 说明 |

| sequence_type | string | 是 | 序列类型 (welcome/sales/retention) |
| emails | number | 是 | 邮件数量 |
| goal | string | 是 | 目标 (conversion/engagement/retention) |
