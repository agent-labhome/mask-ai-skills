# 视频脚本助手

## 功能

专业视频脚本生成工具，短视频、带货、直播全覆盖。

## 使用场景

### 1. 短视频脚本

**输入:** 主题 + 时长

**输出:**
- 分镜脚本
- 口播文案
- 背景音乐建议

### 2. 带货脚本

**输入:** 产品 + 平台

**输出:**
- 开场钩子
- 产品卖点
- 行动号召
- 逼单话术

### 3. 直播话术

**输入:** 直播类型

**输出:**
- 开场话术
- 互动话术
- 促单话术
- 结束话术

### 4. 口播文案

**输入:** 话题 + 风格

**输出:**
- 结构化文案
- 金句建议
- 情绪节奏

## 参数

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| task | string | 是 | 任务类型 |
| platform | string | 是 | 平台 |
| topic | string | 是 | 主题 |
| duration | string | 否 | 时长 |
