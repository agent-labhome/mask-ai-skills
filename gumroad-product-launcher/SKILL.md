# Gumroad产品发布助手

## 功能

帮助创作者在Gumroad上创建、定价、发布数字产品。

## 使用场景

### 1. 产品规划
- 定价策略
- 捆绑销售
- 套餐设计

### 2. Landing Page优化
- 销售文案
- 功能列表
- 用户证言

### 3. 发布策略
- 预热营销
- _launch邮件
- 社交媒体推广

## 参数

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| product_type | string | 是 | 产品类型 (ebook/course/template/skill) |
| price_point | number | 是 | 价格区间 |
| target_audience | string | 是 | 目标用户 |
