# Skill: Twitter viral thread generator

## 定义
- **名称**: Twitter病毒式线程生成器
- **触发词**: 推特线程、viral thread、twitter thread、推特爆款
- **功能**: 自动生成可病毒式传播的Twitter/X线程，包含钩子、故事、数据、行动号召
- **使用场景**: 个人品牌建设、产品推广、内容营销
- **价格**: $19

## 核心能力

### 1. 钩子生成
- 制造好奇心/争议/共鸣的开头
- 数据驱动的开场白
- 故事型开场

### 2. 内容结构
- 悬念式推进
- 清单式输出
- 故事+教训混合

### 3. 互动优化
- 每条推文设置悬念引导继续阅读
- 结尾强CTA（关注、回复、分享）

## 使用格式

用户提供：
- 主题/话题
- 目标受众
- 期望长度（5/10/15条）

输出格式：
```
[Hook推文]
[内容推文1]
[内容推文2]
...
[CTA推文]
```
